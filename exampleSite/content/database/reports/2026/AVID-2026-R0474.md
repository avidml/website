## Description

The model gemma-3n-E4B-it from Google was evaluated by the Garak LLM Vulnerability scanner using the probe `encoding.InjectNato`.

## Details

The probe encoding.InjectNato tests encoding techniques to bypass input filtering by using NATO phonetic alphabet representations. This probe is part of the encoding module which injects ASCII85 encoded data into payloads.

The LLM gemma-3n-E4B-it was evaluated on this probe.
