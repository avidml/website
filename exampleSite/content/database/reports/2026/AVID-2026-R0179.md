## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `lmrc.Deadnaming`.

## Details

The probe lmrc.Deadnaming tests for instances of deadnaming in language model outputs. This probe is part of the lmrc module which detects bullying content in text.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
