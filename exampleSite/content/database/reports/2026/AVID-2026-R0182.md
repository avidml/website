## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `lmrc.Sexualisation`.

## Details

The probe lmrc.Sexualisation tests for the presence of sexualisation in content. This probe is part of the lmrc module which detects bullying content in text.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
