## Description

FPE in paddle.amin (CVE-2023-52308)

## Details

FPE in paddle.amin in PaddlePaddle before 2.6.0. This flaw can cause a runtime crash and a denial of service.

Reason for inclusion in AVID: CVE-2023-52308 describes a finite precision error (FPE) in PaddlePaddle's paddle.amin that can cause a runtime crash and denial of service. PaddlePaddle is a widely used AI framework in general-purpose AI stacks, so this is a software vulnerability in a component used to build/train/deploy AI systems. It constitutes a software supply-chain issue for AI software stacks. The CVE and advisory provide explicit vulnerability behavior (DoS via crash/divide-by-zero) and concrete references, offering sufficient signal for AVID curation.
