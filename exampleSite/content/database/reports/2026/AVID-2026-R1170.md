## Description

Vulnerability CVE-2022-46741

## Details

Out-of-bounds read in gather_tree in PaddlePaddle before 2.4.

Reason for inclusion in AVID: CVE-2022-46741 describes an out-of-bounds read in PaddlePaddle's gather_tree before 2.4. PaddlePaddle is a major ML framework used in AI systems and pipelines. This is a software vulnerability in a library that is part of the AI software stack (dependencies/runtime for AI apps), not hardware/firmware. It affects the security/safety of AI software deployments and has clear CVE and advisory references, providing sufficient evidence for curation.
