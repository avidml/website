## Description

Stack Exhaustion In Tensorflow Serving (CVE-2025-0649)

## Details

Incorrect JSON input stringification in Google's Tensorflow serving versions up to 2.18.0 allows for potentially unbounded recursion leading to server crash.

Reason for inclusion in AVID: CVE-2025-0649 describes a stack exhaustion vulnerability in TensorFlow Serving (AI model serving stack) that can cause a server crash due to unbounded recursion. This is a software vulnerability affecting an AI deployment component used in general-purpose AI systems, and it concerns the software supply chain (dependencies/serving stack). The report provides CVE details and a commit reference as evidence.
