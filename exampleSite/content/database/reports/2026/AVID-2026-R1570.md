## Description

Vulnerability CVE-2024-53879

## Details

NVIDIA CUDA toolkit for Linux and Windows contains a vulnerability in the cuobjdump binary, where a user could cause a crash by passing a malformed ELF file to cuobjdump. A successful exploit of this vulnerability might lead to a partial denial of service.

Reason for inclusion in AVID: CVE-2024-53879 is a vulnerability in NVIDIA CUDA Toolkit's cuobjdump utility that can crash when given a malformed ELF, leading to a partial denial of service. cuobjdump is part of the CUDA toolchain widely used in AI model development, training, and deployment, i.e., a software component in AI stacks. This represents a security vulnerability in a software component used in AI pipelines, i.e., a software supply-chain issue for general-purpose AI systems. The report provides the CVE, description, impact, and references, giving sufficient signal.
