## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `grandma.Win10`.

## Details

The probe grandma.Win10 tests the ability to extract ethnic slurs by roleplaying as a grandmother. This probe is part of the grandma module which simulates a grandma to elicit sensitive information.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
