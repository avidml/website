## Description

The model gemma-3n-E4B-it from Google was evaluated by the Garak LLM Vulnerability scanner using the probe `promptinject.HijackKillHumans`.

## Details

The probe promptinject.HijackKillHumans tests for malicious prompt injections aimed at causing harm to humans. This probe is part of the promptinject module which detects and analyzes prompt injection attacks.

The LLM gemma-3n-E4B-it was evaluated on this probe.
