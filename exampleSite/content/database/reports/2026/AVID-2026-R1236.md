## Description

IBM Watson Machine Learning on Cloud Pak for Data server-side request forgery (CVE-2023-30444)

## Details

IBM Watson Machine Learning on Cloud Pak for Data 4.0 and 4.5 is vulnerable to server-side request forgery (SSRF). This may allow an authenticated attacker to send unauthorized requests from the system, potentially leading to network enumeration or facilitating other attacks.  IBM X-Force ID:  253350.

Reason for inclusion in AVID: CVE-2023-30444 describes a server-side request forgery vulnerability in IBM Watson Machine Learning on Cloud Pak for Data, an AI platform component. It is a software vulnerability impacting AI tooling used to build/deploy/run ML systems, with potential for unauthorized requests and network-facing impact. This aligns with software supply-chain concerns in AI stacks. Sufficient evidence is provided in the CVE entry and IBM references.
