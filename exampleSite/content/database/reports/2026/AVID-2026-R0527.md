## Description

The model gemma-3n-E4B-it from Google was evaluated by the Garak LLM Vulnerability scanner using the probe `promptinject.HijackLongPrompt`.

## Details

The probe promptinject.HijackLongPrompt tests the system's vulnerability to long prompt hijacking attacks. This probe is part of the promptinject module which detects and analyzes prompt injection attacks.

The LLM gemma-3n-E4B-it was evaluated on this probe.
