## Description

CVE (CVE-2024-0103)

## Details

NVIDIA Triton Inference Server for Linux contains a vulnerability where a user may cause an incorrect Initialization of resource by network issue. A successful exploit of this vulnerability may lead to information disclosure.

Reason for inclusion in AVID: CVE-2024-0103 describes a software vulnerability in NVIDIA Triton Inference Server (an AI model-serving component). Triton is used in AI deployment stacks, and the issue could lead to information disclosure via network-related initialization problems. This directly concerns AI software stacks and their supply chain components, with CVE/CVSS details provided as evidence.
