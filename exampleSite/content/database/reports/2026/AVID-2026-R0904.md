## Description

Heap OOB read in `tf.ragged.cross` (CVE-2021-41212)

## Details

TensorFlow is an open source platform for machine learning. In affected versions the shape inference code for `tf.ragged.cross` can trigger a read outside of bounds of heap allocated array. The fix will be included in TensorFlow 2.7.0. We will also cherrypick this commit on TensorFlow 2.6.1, TensorFlow 2.5.2, and TensorFlow 2.4.4, as these are also affected and still in supported range.

Reason for inclusion in AVID: CVE-2021-41212 describes a heap out-of-bounds read in TensorFlow (tf.ragged.cross), a vulnerability in a software library used to develop and run AI/ML models. It affects TensorFlow versions used in AI pipelines and is a conventional CVE with public advisories and references. It relates to software components (ML framework) critical to general-purpose AI systems, not hardware/firmware. The report provides clear evidence (NVD entry, GitHub advisory, commit) and CVSS data, indicating a genuine security vulnerability in the software stack used for AI—thus relevant to the AI supply chain.
