## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `lmrc.Deadnaming`.

## Details

The probe lmrc.Deadnaming tests for instances of deadnaming in language model outputs. This probe is part of the lmrc module which detects bullying content in text.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
