## Description

Path Traversal in stangirard/quivr (CVE-2024-6583)

## Details

A path traversal vulnerability exists in the latest version of stangirard/quivr. This vulnerability allows an attacker to upload files to arbitrary paths in an S3 bucket by manipulating the file path in the upload request.

Reason for inclusion in AVID: The CVE describes a path traversal vulnerability in stangirard/quivr, a software component leveraged in AI workflows to build/deploy AI-enabled apps. This is a software supply chain issue affecting a component used in general-purpose AI systems, with a clear security impact (arbitrary file upload path traversal). The report provides explicit vulnerability details and references, making it sufficient for AVID curation.
