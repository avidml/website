## Description

Vulnerability CVE-2024-36328

## Details

Integer overflow within AMD NPU Driver could allow a local attacker to write out of bounds, potentially leading to loss of integrity or availability.

Reason for inclusion in AVID: CVE-2024-36328 describes an integer overflow in the AMD NPU (neural processing unit) driver, which is a software component used to accelerate AI workloads (Ryzen AI Software). This is a vulnerability in a software/drivers layer that is part of the stack used to build, run, and accelerate general-purpose AI systems. It has potential to impact integrity/availability and is exploitable locally, aligning with typical security risks in AI software pipelines. As the issue concerns a software component (driver) used in AI stacks, it qualifies for AVID curation under the general AI supply chain scope.
