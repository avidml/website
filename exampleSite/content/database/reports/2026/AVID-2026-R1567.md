## Description

Vulnerability CVE-2024-53876

## Details

NVIDIA CUDA toolkit for all platforms contains a vulnerability in the nvdisasm binary, where a user could cause an out-of-bounds read by passing a malformed ELF file to nvdisasm. A successful exploit of this vulnerability might lead to a partial denial of service.

Reason for inclusion in AVID: CVE-2024-53876 describes a software vulnerability in the CUDA Toolkit (nvdisasm) that allows an out-of-bounds read with potential DoS when processing a malformed ELF file. CUDA Toolkit components are widely used in AI/ML software stacks (drivers/libraries for model training, inference, and deployment). This is a software supply-chain issue affecting AI software components, not a hardware/firmware-only vulnerability. The CVE entry provides sufficient signal (CVE ID, description, and references) to categorize it as a security vulnerability in a software dependency used in AI systems.
