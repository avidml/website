## Description

Vulnerability CVE-2024-0076

## Details

NVIDIA CUDA toolkit for all platforms contains a vulnerability in cuobjdump and nvdisasm where an attacker may cause a crash by tricking a user into reading a malformed ELF file. A successful exploit of this vulnerability may lead to a partial denial of service.

Reason for inclusion in AVID: CVE-2024-0076 describes a vulnerability in NVIDIA CUDA Toolkit tools (cuobjdump and nvdisasm) that can crash when processing a malformed ELF, leading to a partial DoS. The CUDA Toolkit is a core component used in building, deploying, and running AI software stacks (GPU drivers, runtimes, and development tools). Therefore, this is a software supply-chain vulnerability within AI pipelines and tooling, with clear security implications.
