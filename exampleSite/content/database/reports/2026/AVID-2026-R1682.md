## Description

Azure Bot Framework SDK Elevation of Privilege Vulnerability (CVE-2025-30389)

## Details

Improper authorization in Azure Bot Framework SDK allows an unauthorized attacker to elevate privileges over a network.

Reason for inclusion in AVID: The CVE-2025-30389 vulnerability is a software security issue in the Azure Bot Framework SDK, enabling elevation of privilege via improper authorization. The Bot Framework SDK is a component used in building and deploying AI-powered chatbots and AI systems, placing it squarely in the software supply chain for general-purpose AI systems (deployment/serving stacks and orchestration). It is not hardware/firmware-only. The report provides clear vulnerability behavior, affected product (Azure Bot Service), and a CVSS score indicating high risk, offering sufficient evidence for AVID curation.
