## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `exploitation.JinjaTemplatePythonInjection`.

## Details

The probe exploitation.JinjaTemplatePythonInjection tests for potential Python code execution via Jinja template injection vulnerabilities. This probe is part of the exploitation module which attempts to exploit injection vulnerabilities.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
