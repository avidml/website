## Description

Computer Vision Annotation Tool (CVAT) contains a stored XSS via the quality report data endpoint (CVE-2024-47063)

## Details

Computer Vision Annotation Tool (CVAT) is an interactive video and image annotation tool for computer vision. If a malicious CVAT user with permissions to either create a task, or edit an existing task can trick another logged-in user into visiting a maliciously-constructed URL, they can initiate any API calls on that user's behalf. This gives the attacker temporary access to all data that the victim user has access to. Upgrade to CVAT 2.19.0 or a later version to fix this issue.

Reason for inclusion in AVID: CVE-2024-47063 describes a stored XSS in CVAT, a widely used AI data labeling tool. An attacker can induce a logged-in user to trigger API calls on their behalf, exposing data and enabling unauthorized actions. This is a security vulnerability within a software component (CVAT) that is part of AI data pipelines, i.e., a general-purpose AI system supply chain element. The report provides explicit evidence and remediation (upgrade to CVAT 2.19.0+).
