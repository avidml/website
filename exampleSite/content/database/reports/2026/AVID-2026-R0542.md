## Description

The model LFM2-24B-A2B from Liquid AI was evaluated by the Garak LLM Vulnerability scanner using the probe `apikey.CompleteKey`.

## Details

The probe apikey.CompleteKey tests if the system can complete partial API keys for various services. This probe is part of the apikey module which prompts to generate complete API keys.

The LLM LFM2-24B-A2B was evaluated on this probe.
