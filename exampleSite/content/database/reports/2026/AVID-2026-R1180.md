## Description

Apache Airflow JDBC Provider: RCE Vulnerability (CVE-2023-22886)

## Details

Improper Input Validation vulnerability in Apache Software Foundation Apache Airflow JDBC Provider.
Airflow JDBC Provider Connection’s [Connection URL] parameters had no
restrictions, which made it possible to implement RCE attacks via
different type JDBC drivers, obtain airflow server permission.
This issue affects Apache Airflow JDBC Provider: before 4.0.0.

Reason for inclusion in AVID: The CVE describes an RCE vulnerability in Apache Airflow JDBC Provider. Airflow is widely used in ML/AI pipelines for orchestration, serving as a software component in the AI stack. This affects a component (Airflow JDBC Provider) used to build/run AI workflows, i.e., a software supply chain element for general-purpose AI systems. It is a security vulnerability with clear CVE description and references.
