## Description

Vulnerability CVE-2024-24590

## Details

Deserialization of untrusted data can occur in versions 0.17.0 to 1.14.2 of the client SDK of Allegro AI’s ClearML platform, enabling a maliciously uploaded artifact to run arbitrary code on an end user’s system when interacted with.

Reason for inclusion in AVID: CVE-2024-24590 describes a deserialization vulnerability in Allegro AI ClearML client SDK (versions 0.17.0–1.14.2) that enables arbitrary code execution via a maliciously uploaded artifact. This directly concerns software used in AI/ML workflows (AI tooling, model management, and MLOps pipelines) and impacts the software supply chain of general-purpose AI systems. It is a security vulnerability (RCE) with explicit evidence in CVSS/CWE entries.
