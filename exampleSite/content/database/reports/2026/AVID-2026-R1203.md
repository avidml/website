## Description

TensorFlow has segmentation fault in tfg-translate (CVE-2023-25671)

## Details

TensorFlow is an open source platform for machine learning. There is out-of-bounds access due to mismatched integer type sizes. A fix is included in TensorFlow version 2.12.0 and version 2.11.1.

Reason for inclusion in AVID: CVE-2023-25671 describes a software vulnerability in TensorFlow (an AI framework) causing segmentation fault via out-of-bounds access. This directly concerns AI/ML systems and TensorFlow usage in ML pipelines, representing a vulnerability in a component commonly used to build/deploy AI systems. The report provides enough evidence (CVE entry, advisory, fixes) to classify as a security vulnerability affecting the AI software stack and its supply chain.
