## Description

Vulnerability CVE-2024-53875

## Details

NVIDIA CUDA toolkit for all platforms contains a vulnerability in the cuobjdump binary, where a user could cause an out-of-bounds read by passing a malformed ELF file to cuobjdump. A successful exploit of this vulnerability might lead to a partial denial of service.

Reason for inclusion in AVID: CVE-2024-53875 describes a software vulnerability in NVIDIA CUDA toolkit's cuobjdump that can cause an out-of-bounds read leading to a partial denial of service when processing a malformed ELF file. The CUDA toolkit is a core dependency/tooling used to build and run AI models and pipelines, which places this vulnerability squarely in the software supply chain for general-purpose AI systems. The issue is reported as a CVE with a defined vulnerability behavior (DoS). The report provides explicit signals (CVE entry, description, CVSS, references).
