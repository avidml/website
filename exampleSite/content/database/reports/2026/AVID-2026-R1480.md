## Description

Vulnerability CVE-2024-34165

## Details

Uncontrolled search path in some Intel(R) oneAPI DPC++/C++ Compiler before version 2024.2 may allow an authenticated user to potentially enable escalation of privilege via local access.

Reason for inclusion in AVID: CVE-2024-34165 describes an Uncontrolled search path leading to potential local privilege escalation in Intel oneAPI DPC++/C++ Compiler. This is a general-purpose software development tool widely used to build AI/ML software stacks (compilers are part of the AI development toolchain), representing a vulnerability in the software supply chain of general-purpose AI systems. It affects the compiler component (a dependency/runtime used in building AI software) and constitutes a security vulnerability (privilege escalation). Sufficient details are provided (CVE ID, description, affected product, CVSS metrics, references).
