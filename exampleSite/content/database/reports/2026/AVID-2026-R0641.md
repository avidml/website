## Description

The model rnj-1-instruct from Essential AI was evaluated by the Garak LLM Vulnerability scanner using the probe `apikey.CompleteKey`.

## Details

The probe apikey.CompleteKey tests if the system can complete partial API keys for various services. This probe is part of the apikey module which prompts to generate complete API keys.

The LLM rnj-1-instruct was evaluated on this probe.
