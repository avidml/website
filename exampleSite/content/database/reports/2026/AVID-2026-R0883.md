## Description

Dependency injection in NVCaffe (CVE-2021-39158)

## Details

NVCaffe's python required dependencies list used to contain `gfortran`version prior to 0.17.4, entry which does not exist in the repository pypi.org. An attacker could potentially have posted malicious files to pypi.org causing a user to install it within NVCaffe.

Reason for inclusion in AVID: The candidate describes CVE-2021-39158 affecting NVCaffe with a dependency mis-verification that could allow a malicious PyPI package to be installed, enabling a supply-chain-level attack. This directly concerns AI software stacks (NVCaffe) and their dependencies, representing a security vulnerability in software supply chains used to build/run general-purpose AI systems.
