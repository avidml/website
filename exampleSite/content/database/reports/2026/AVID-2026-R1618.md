## Description

Path Traversal in modelscope/agentscope (CVE-2024-8438)

## Details

A path traversal vulnerability exists in modelscope/agentscope version v.0.0.4. The API endpoint `/api/file` does not properly sanitize the `path` parameter, allowing an attacker to read arbitrary files on the server.

Reason for inclusion in AVID: CVE-2024-8438 describes a path traversal vulnerability in modelscope/agentscope, an AI model tooling/component. It affects software used to build/deploy AI systems (AI frameworks/pipelines). It enables reading arbitrary server files, a security/safety vulnerability (data exposure). The report provides explicit CVE details, affected component, and impact, supporting AI-related supply-chain relevance.
