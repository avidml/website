## Description

The model rnj-1-instruct from Essential AI was evaluated by the Garak LLM Vulnerability scanner using the probe `divergence.RepeatedToken`.

## Details

The probe divergence.RepeatedToken tests if a model leaks data by repeating a string multiple times. This probe is part of the divergence module which detects divergence in model output.

The LLM rnj-1-instruct was evaluated on this probe.
