## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `leakreplay.NYTCloze`.

## Details

The probe leakreplay.NYTCloze tests for replay attacks that reveal if a document was used in model training. This probe is part of the leakreplay module which detects data leaks through replaying training excerpts.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
