## Description

Segfault in paddle.dot (CVE-2023-38676)

## Details

Nullptr in paddle.dot in PaddlePaddle before 2.6.0. This flaw can cause a runtime crash and a denial of service.

Reason for inclusion in AVID: The CVE describes a null pointer dereference in PaddlePaddle's paddle.dot, causing a runtime crash and potential DoS. PaddlePaddle is a widely-used ML framework, so this vulnerability affects AI models and pipelines. It resides in a software component that is part of the AI software supply chain (framework/dependency). It is clearly a security vulnerability (CWE-476) with CVSS context provided. Sufficient evidence is present in the report and references.
