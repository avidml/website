## Description

TensorFlow has Floating Point Exception in TensorListSplit with XLA (CVE-2023-25673)

## Details

TensorFlow is an open source platform for machine learning. Versions prior to 2.12.0 and 2.11.1 have a Floating Point Exception in TensorListSplit with XLA. A fix is included in TensorFlow version 2.12.0 and version 2.11.1.

Reason for inclusion in AVID: CVE-2023-25673 describes a vulnerability in TensorFlow (a core ML framework) causing a floating point exception in TensorListSplit with XLA in affected versions. This is a software vulnerability in a widely used ML stack, impacting AI systems built on TensorFlow. It affects software supply chain components (TensorFlow library/dependencies) used to train/serve AI systems. It is security-related (availability impact) with CVSS high, and there is explicit evidence in CVE/NVD references. Sufficient evidence from CVE entry and vendor advisory links.
