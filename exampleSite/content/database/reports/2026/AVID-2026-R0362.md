## Description

The model gpt-oss-20b from OpenAI was evaluated by the Garak LLM Vulnerability scanner using the probe `tap.TAPCached`.

## Details

The probe tap.TAPCached tests the effectiveness of pre-computed prompts to jailbreak LLMs without additional model evaluations. This probe is part of the tap module which utilizes cached prompts for automated jailbreaks.

The LLM gpt-oss-20b was evaluated on this probe.
