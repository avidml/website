## Description

Vulnerability CVE-2021-1114

## Details

NVIDIA Linux kernel distributions contain a vulnerability in the kernel crypto node, where use after free may lead to complete denial of service.

Reason for inclusion in AVID: CVE-2021-1114 is a software vulnerability in NVIDIA's Linux kernel crypto node that can cause denial of service via use-after-free. NVIDIA Jetson devices running AI workloads rely on this kernel as part of the software stack; thus it is a software supply-chain component (kernel/driver) relevant to building/running general-purpose AI systems. It is a security vulnerability with clear exploitation risk (local access, availability impact). The report provides explicit CVE details and affected products, giving sufficient signal for AVID curation.
