## Description

Vulnerability CVE-2023-25517

## Details

NVIDIA vGPU software contains a vulnerability in the Virtual GPU Manager (vGPU plugin), where a guest OS may be able to control resources for which it is not authorized, which may lead to information disclosure and data tampering.

Reason for inclusion in AVID: CVE-2023-25517 describes a vulnerability in NVIDIA vGPU software (vGPU plugin) that allows a guest OS to access resources it is not authorized to control, leading to information disclosure and data tampering. This is a software component in the GPU virtualization stack used to run AI workloads, i.e., part of the software supply chain for general-purpose AI systems. It is a security vulnerability with local access requirements.
