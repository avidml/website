## Description

SQL Injection in mintplex-labs/anything-llm (CVE-2023-4899)

## Details

SQL Injection in GitHub repository mintplex-labs/anything-llm prior to 0.0.1.

Reason for inclusion in AVID: The CVE describes a SQL injection vulnerability (CWE-89) in an AI-related software project (mintplex-labs/anything-llm) that would be used in AI model deployment/serving pipelines. It affects the integrity and confidentiality of data, has CVSS high severity, and concerns software components used in building/running general-purpose AI systems (the artifact itself). The report provides explicit evidence (CVE, NVD entry, commit link) supporting this vulnerability in a software artifact central to AI stacks. Therefore it should be kept for AVID curation as a software supply chain vulnerability in AI systems.
