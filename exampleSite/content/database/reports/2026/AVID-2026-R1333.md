## Description

Path Traversal: '\..\filename' in mlflow/mlflow (CVE-2023-6831)

## Details

Path Traversal: '\..\filename' in GitHub repository mlflow/mlflow prior to 2.9.2.

Reason for inclusion in AVID: CVE-2023-6831 is a path traversal vulnerability in mlflow<2.9.2. MLflow is a core AI tooling/framework used in building, training, deploying AI models, making this a software supply-chain issue in AI systems. It is a security vulnerability with documented impact, and the report provides sufficient signals (CVE, affected version, commit).
