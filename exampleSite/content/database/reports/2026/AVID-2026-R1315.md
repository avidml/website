## Description

Null pointer dereference in paddle.crop (CVE-2023-52312)

## Details

Nullptr dereference in paddle.crop in PaddlePaddle before 2.6.0. This flaw can cause a runtime crash and a denial of service.

Reason for inclusion in AVID: CVE-2023-52312 describes a null pointer dereference in PaddlePaddle's paddle.crop, causing runtime crashes and potential DoS. PaddlePaddle is a general-purpose AI framework; vulnerabilities in such libraries affect AI model training, inference, and deployment pipelines. This is clearly a software supply-chain component (library/dependency) used to build/run AI systems, not a hardware/firmware issue. The report provides explicit CVE details, affected component, impact, and references, satisfying evidence requirements.
