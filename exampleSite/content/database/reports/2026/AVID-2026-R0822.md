## Description

Deleting PRTBs associated to a group doesn't cause deletion of corresponding RoleBindings (CVE-2021-36775)

## Details

a Improper Access Control vulnerability in SUSE Rancher allows users to keep privileges that should have been revoked. This issue affects: SUSE Rancher Rancher versions prior to 2.4.18; Rancher versions prior to 2.5.12; Rancher versions prior to 2.6.3.

Reason for inclusion in AVID: The CVE describes an improper access control flaw in SUSE Rancher where deleting PRTBs does not delete corresponding RoleBindings, enabling privilege retention. Rancher is a software component commonly used to deploy and manage containerized workloads, including AI/ML pipelines. This places the vulnerability within the software stack that underpins general-purpose AI systems (deployment/orchestration, runtimes, CI/CD, etc.). It is a security/vulnerability issue with clear CVE details and references, providing sufficient evidence for AVID curation.
