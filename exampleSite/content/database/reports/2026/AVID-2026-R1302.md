## Description

WordPress AI Engine plugin <= 1.9.98 - Unauthenticated Arbitrary File Upload vulnerability (CVE-2023-51409)

## Details

Unrestricted Upload of File with Dangerous Type vulnerability in Jordy Meow AI Engine: ChatGPT Chatbot.This issue affects AI Engine: ChatGPT Chatbot: from n/a through 1.9.98.

Reason for inclusion in AVID: CVE-2023-51409 describes an unauthenticated arbitrary file upload in the WordPress AI Engine plugin (AI Engine: ChatGPT Chatbot). This is a software vulnerability in a component used to deploy AI features on websites, which constitutes a software supply chain element in AI deployments. The flaw enables improper file uploads with potential remote code execution, impacting security and safety of AI-powered web deployments. The report provides explicit vulnerability behavior, affected version range, and CVE reference.
