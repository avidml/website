## Description

Vulnerability CVE-2024-0115

## Details

NVIDIA CV-CUDA for Ubuntu 20.04, Ubuntu 22.04, and Jetpack contains a vulnerability in Python APIs where a user may cause an uncontrolled resource consumption issue by a long running CV-CUDA Python process. A successful exploit of this vulnerability may lead to denial of service and data loss.

Reason for inclusion in AVID: CVE-2024-0115 describes a vulnerability in NVIDIA CV-CUDA Python APIs that can cause uncontrolled resource consumption, leading to denial of service and potential data loss. This is a software component used in AI workflows (CV-CUDA) and is part of the AI software stack, making it relevant to AI systems and their supply chains. The issue affects a software dependency/runtimes used to build/deploy AI capabilities, which aligns with software supply-chain concerns rather than hardware/firmware-only problems. The CVE entry provides explicit vulnerability behavior and impact signals, meeting criteria for AVID curation.
