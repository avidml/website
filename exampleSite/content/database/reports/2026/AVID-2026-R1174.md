## Description

Vulnerability CVE-2023-0193

## Details

NVIDIA CUDA Toolkit SDK contains a vulnerability in cuobjdump, where a local user running the tool against a malicious binary may cause an out-of-bounds read, which may result in a limited denial of service and limited information disclosure.

Reason for inclusion in AVID: CVE-2023-0193 describes an out-of-bounds read in NVIDIA CUDA Toolkit's cuobjdump that can be triggered locally by processing a crafted binary, causing a limited DoS and information disclosure. CUDA Toolkit is a software stack commonly used in AI model development and deployment, so this is a software supply chain vulnerability affecting AI tooling and pipelines. It concerns a genuine vulnerability (CWE-125) with CVSS metrics provided. Therefore it qualifies for AVID curation.
