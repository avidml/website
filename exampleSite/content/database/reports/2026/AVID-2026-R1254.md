## Description

Azure Identity SDK Remote Code Execution Vulnerability (CVE-2023-36414)

## Details

Azure Identity SDK Remote Code Execution Vulnerability

Reason for inclusion in AVID: CVE-2023-36414 is a remote code execution vulnerability in the Azure Identity SDK for .NET. This SDK is a common dependency in AI deployment stacks on Azure (authentication/authorization for AI services, storage, compute). As a vulnerability in a library used to build/run AI systems, it represents a software supply-chain risk for general-purpose AI workflows. It has clear security impact (RCE) with public references (NVD, MSRC). Therefore it should be kept for AVID curation.
