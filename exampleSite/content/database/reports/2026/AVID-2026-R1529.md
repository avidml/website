## Description

DeepSpeed Remote Code Execution Vulnerability (CVE-2024-43497)

## Details

DeepSpeed Remote Code Execution Vulnerability

Reason for inclusion in AVID: CVE-2024-43497 describes a remote code execution vulnerability in DeepSpeed, a library widely used in AI model training and deployment stacks. This is a software supply chain issue within AI software packages/frameworks, not hardware/firmware. It directly affects components used to build/train/run general-purpose AI systems, and exposes security risk (RCE). Evidence includes CVE ID, description of RCE, and CWE-77 (command injection).
