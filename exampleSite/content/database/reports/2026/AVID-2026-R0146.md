## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `encoding.InjectMorse`.

## Details

The probe encoding.InjectMorse tests the injection of Morse code to bypass input filtering safeguards. This probe is part of the encoding module which injects ASCII85 encoded data into payloads.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
