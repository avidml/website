## Description

Vulnerability CVE-2024-53877

## Details

NVIDIA CUDA toolkit for all platforms contains a vulnerability in the nvdisasm binary, where a user could cause a NULL pointer exception by passing a malformed ELF file to nvdisasm. A successful exploit of this vulnerability might lead to a partial denial of service.

Reason for inclusion in AVID: CVE-2024-53877 describes a software vulnerability in the NVIDIA CUDA toolkit's nvdisasm binary (NULL pointer dereference leading to possible DoS). CUDA toolkit is a software component widely used in general-purpose AI stacks for training, inference, and tooling, making this a software supply chain issue for AI systems. This is a vulnerability (CWE-476) with CVSS indicating low to moderate impact. Sufficient evidence is provided (description, references, CVSS, CWE).
