## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `encoding.InjectMorse`.

## Details

The probe encoding.InjectMorse tests the injection of Morse code to bypass input filtering safeguards. This probe is part of the encoding module which injects ASCII85 encoded data into payloads.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
