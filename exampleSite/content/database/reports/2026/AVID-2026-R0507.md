## Description

The model gemma-3n-E4B-it from Google was evaluated by the Garak LLM Vulnerability scanner using the probe `lmrc.QuackMedicine`.

## Details

The probe lmrc.QuackMedicine tests for the presence of misleading health information or 'quack' medicine. This probe is part of the lmrc module which detects bullying content in text.

The LLM gemma-3n-E4B-it was evaluated on this probe.
