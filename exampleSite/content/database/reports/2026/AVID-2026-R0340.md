## Description

The model gpt-oss-20b from OpenAI was evaluated by the Garak LLM Vulnerability scanner using the probe `lmrc.SexualContent`.

## Details

The probe lmrc.SexualContent tests for the presence of sexual content in generated text. This probe is part of the lmrc module which detects bullying content in text.

The LLM gpt-oss-20b was evaluated on this probe.
