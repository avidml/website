## Description

TensorFlow vulnerable to Out-of-Bounds Read in GRUBlockCellGrad (CVE-2023-25658)

## Details

TensorFlow is an open source platform for machine learning. Prior to versions 2.12.0 and 2.11.1, an out of bounds read is in GRUBlockCellGrad. A fix is included in TensorFlow 2.12.0 and 2.11.1.

Reason for inclusion in AVID: The CVE describes a security vulnerability in TensorFlow (an AI framework) causing an out-of-bounds read. TensorFlow is a core component used to build/train/deploy ML systems, so it is AI-related and part of the software supply chain for general-purpose AI. The issue is a CVE-style vulnerability with defined impact and a fix, providing sufficient signal for curation.
