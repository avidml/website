## Description

FPE in paddle.nanmedian (CVE-2023-38674)

## Details

FPE in paddle.nanmedian in PaddlePaddle before 2.6.0. This flaw can cause a runtime crash and a denial of service.

Reason for inclusion in AVID: This CVE describes a runtime vulnerability (divide-by-zero) in PaddlePaddle's nanmedian, a core ML framework library used to build/run AI systems. It affects PaddlePaddle (a dependency) used in AI pipelines, so it's a software supply-chain issue within general-purpose AI software stacks. It is a security vulnerability (causes a Denial of Service), with explicit CVE/NVD references and CVSS details.
