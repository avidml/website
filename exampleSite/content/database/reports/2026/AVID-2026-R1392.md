## Description

DeepFaceLab main.py apply_xseg deserialization (CVE-2024-1432)

## Details

** UNSUPPORTED WHEN ASSIGNED ** A vulnerability was found in DeepFaceLab pretrained DF.wf.288res.384.92.72.22 and classified as problematic. This issue affects the function apply_xseg of the file main.py. The manipulation leads to deserialization. The attack may be initiated remotely. The complexity of an attack is rather high. The exploitation is known to be difficult. The exploit has been disclosed to the public and may be used. The associated identifier of this vulnerability is VDB-253391. NOTE: This vulnerability only affects products that are no longer supported by the maintainer.

Reason for inclusion in AVID: CVE-2024-1432 describes a deserialization vulnerability in DeepFaceLab (an AI tool used in ML workflows for deepfake generation). This affects a software component used to build/run AI workflows, and presents a security vulnerability (CWE-502). The report provides explicit vulnerability details (deserialization in apply_xseg, potential remote initiation, CVSS/ CWE references), supporting its relevance to the AI software supply chain. Hardware/firmware-only vulnerabilities are not involved. Therefore, it qualifies for AVID curation as a vulnerability in the supply chain of general-purpose AI systems.
