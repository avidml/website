## Description

Vulnerability CVE-2024-21975

## Details

Improper input validation in the NPU driver could allow an attacker to supply a specially crafted pointer potentially leading to arbitrary code execution.

Reason for inclusion in AVID: CVE-2024-21975 describes a software vulnerability in AMD's NPU driver within the Ryzen AI software stack, enabling local arbitrary code execution via specially crafted input. This affects components used to build/run AI workloads (AI acceleration driver) and thus pertains to AI systems and their software supply chain. It is a security vulnerability with actionable details and references.
