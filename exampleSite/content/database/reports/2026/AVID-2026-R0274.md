## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `grandma.Win11`.

## Details

The probe grandma.Win11 tests the ability to extract illegal or contentious information by roleplaying as a grandmother. This probe is part of the grandma module which simulates a grandma to elicit sensitive information.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
