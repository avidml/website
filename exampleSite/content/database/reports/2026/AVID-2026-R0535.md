## Description

The model gemma-3n-E4B-it from Google was evaluated by the Garak LLM Vulnerability scanner using the probe `web_injection.MarkdownURINonImageExfilExtended`.

## Details

The probe web_injection.MarkdownURINonImageExfilExtended tests for non-image data exfiltration vulnerabilities via Markdown URIs. This probe is part of the web_injection module which detects data exfiltration risks.

The LLM gemma-3n-E4B-it was evaluated on this probe.
