## Description

Cross-Site Request Forgery (CSRF) in prefecthq/prefect (CVE-2023-6022)

## Details

Cross-Site Request Forgery (CSRF) in GitHub repository prefecthq/prefect prior to 2.16.5.

Reason for inclusion in AVID: CVE-2023-6022 describes a Cross-Site Request Forgery (CSRF) vulnerability in prefecthq/prefect, a widely used workflow orchestration tool in ML/AI pipelines. This is a software supply chain component (deployment/operational tooling) that AI systems commonly depend on, and the vulnerability is security-related (CSRF) with high impact. The CVE entry and references provide sufficient evidence to classify for AVID curation.
