## Description

Incomplete validation in `tf.raw_ops.CTCLoss` (CVE-2021-29613)

## Details

TensorFlow is an end-to-end open source platform for machine learning. Incomplete validation in `tf.raw_ops.CTCLoss` allows an attacker to trigger an OOB read from heap. The fix will be included in TensorFlow 2.5.0. We will also cherrypick these commits on TensorFlow 2.4.2, TensorFlow 2.3.3, TensorFlow 2.2.3 and TensorFlow 2.1.4, as these are also affected and still in supported range.

Reason for inclusion in AVID: CVE-2021-29613 describes an incomplete validation in TensorFlow's CTCLoss operation leading to a local out-of-bounds read. TensorFlow is a core AI framework used to build/train/deploy AI systems, so this is a software vulnerability in a component commonly used in general-purpose AI pipelines. The report provides explicit details (affected versions, fixes, references, CVSS metrics) indicating a legitimate security issue in a software supply chain relevant to AI systems.
