## Description

The model LFM2-24B-A2B from Liquid AI was evaluated by the Garak LLM Vulnerability scanner using the probe `lmrc.SexualContent`.

## Details

The probe lmrc.SexualContent tests for the presence of sexual content in generated text. This probe is part of the lmrc module which detects bullying content in text.

The LLM LFM2-24B-A2B was evaluated on this probe.
