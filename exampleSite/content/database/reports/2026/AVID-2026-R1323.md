## Description

MLflow Arbitrary File Write (CVE-2023-6018)

## Details

An attacker can overwrite any file on the server hosting MLflow without any authentication.

Reason for inclusion in AVID: CVE-2023-6018 describes an Arbitrary File Write vulnerability in MLflow, allowing an unauthenticated attacker to overwrite files on the server. This is a software vulnerability with clear security impact. MLflow is a core AI lifecycle tool used to build, deploy, and run AI/ML systems, placing the issue squarely in the AI software stack. As MLflow is part of the AI deployment/serving ecosystem, this constitutes a software supply-chain risk for general-purpose AI systems. The report provides explicit vulnerability details (unrestricted file write, no auth) and references, satisfying evidence requirements.
