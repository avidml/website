## Description

Deserialization of Untrusted Data in binary-husky/gpt_academic (CVE-2024-11039)

## Details

A pickle deserialization vulnerability exists in the Latex English error correction plug-in function of binary-husky/gpt_academic versions up to and including 3.83. This vulnerability allows attackers to achieve remote command execution by deserializing untrusted data. The issue arises from the inclusion of numpy in the deserialization whitelist, which can be exploited by constructing a malicious compressed package containing a merge_result.pkl file and a merge_proofread_en.tex file. The vulnerability is fixed in commit 91f5e6b.

Reason for inclusion in AVID: CVE-2024-11039 describes a pickle deserialization vulnerability in binary-husky/gpt_academic that enables remote code execution via untrusted data. The affected artifact is an AI-related software component used within AI workflows/pipelines, constituting a software supply-chain item in general-purpose AI systems. The report provides explicit vulnerability behavior and a fix commit, satisfying evidentiary requirements.
