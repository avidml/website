## Description

Relative Path Traversal in mlflow/mlflow (CVE-2023-2356)

## Details

Relative Path Traversal in GitHub repository mlflow/mlflow prior to 2.3.1.

Reason for inclusion in AVID: CVE-2023-2356 describes a relative path traversal vulnerability in mlflow/mlflow (before 2.3.1). MLflow is a software component widely used in AI/ML pipelines for tracking, packaging, and serving models. This is a software-supply-chain-type issue affecting AI systems, not hardware/firmware. The CVE provides explicit vulnerability behavior (path traversal) and references (NVD, commit), offering sufficient signal for classification.
