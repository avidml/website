## Description

Chain Sea Information Integration Co., Ltd ai chatbot system - Arbitrary File Upload (CVE-2021-44164)

## Details

Chain Sea ai chatbot system’s file upload function has insufficient filtering for special characters in URLs, which allows a remote attacker to by-pass file type validation, upload malicious script and execute arbitrary code without authentication, in order to take control of the system or terminate service.

Reason for inclusion in AVID: CVE-2021-44164 describes an arbitrary file upload vulnerability in an AI chatbot system that allows remote code execution, with high impact (CVSS 9.8). This directly concerns software used in AI deployment/serving stacks and could compromise AI services, making it relevant to the general-purpose AI software supply chain. The advisory clearly indicates a vulnerability in a software component that would be used in AI pipelines, not hardware/firmware-only. Evidence includes CVSS vector, CWE-434, and documented exploitation path.
