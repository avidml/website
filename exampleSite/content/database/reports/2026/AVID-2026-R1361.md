## Description

Vulnerability CVE-2024-0917

## Details

remote code execution in paddlepaddle/paddle 2.6.0

Reason for inclusion in AVID: CVE-2024-0917 describes remote code execution in PaddlePaddle (a major ML framework). This vulnerability affects software used in AI model development, training, and deployment, i.e., general-purpose AI systems. It is in a software component (PaddlePaddle) that AI pipelines rely on, posing a security risk with potential RCE. The report provides explicit CVE reference, affected product, impact details, and CVSS/CWE signals, offering sufficient evidence for curation.
