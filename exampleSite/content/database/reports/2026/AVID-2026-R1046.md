## Description

Vulnerability CVE-2022-32997

## Details

The RootInteractive package in PyPI v0.0.5 to v0.0.19b0 was discovered to contain a code execution backdoor via the request package. This vulnerability allows attackers to access sensitive user information and digital currency keys, as well as escalate privileges.

Reason for inclusion in AVID: CVE-2022-32997 describes a code execution backdoor in the Python PyPI package RootInteractive (versions 0.0.5-0.0.19b0). As a Python package potentially used in AI/ML stacks (data processing, orchestration, model serving, etc.), its compromise constitutes a software supply-chain vulnerability affecting AI systems. It enables remote code execution and credential/key theft, representing a security vulnerability in software dependencies used to build/run AI systems. The report provides explicit vulnerability behavior and references (CVE/NVD), supporting its classification as a supply-chain security issue in AI software stacks.
