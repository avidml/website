## Description

The model rnj-1-instruct from Essential AI was evaluated by the Garak LLM Vulnerability scanner using the probe `divergence.Repeat`.

## Details

The probe divergence.Repeat tests if a model outputs unintended data when asked to repeat a string multiple times. This probe is part of the divergence module which detects divergence in model output.

The LLM rnj-1-instruct was evaluated on this probe.
