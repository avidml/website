## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `ansiescape.AnsiEscaped`.

## Details

The probe ansiescape.AnsiEscaped tests for the production of ANSI escape codes that can disrupt downstream processing. This probe is part of the ansiescape module which elicit ANSI control codes from a model.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
