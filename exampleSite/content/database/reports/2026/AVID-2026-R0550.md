## Description

The model LFM2-24B-A2B from Liquid AI was evaluated by the Garak LLM Vulnerability scanner using the probe `divergence.RepeatedToken`.

## Details

The probe divergence.RepeatedToken tests if a model leaks data by repeating a string multiple times. This probe is part of the divergence module which detects divergence in model output.

The LLM LFM2-24B-A2B was evaluated on this probe.
