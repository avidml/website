## Description

Denial of Service and Arbitrary File Write in h2oai/h2o-3 (CVE-2024-10572)

## Details

In h2oai/h2o-3 version 3.46.0.1, the `run_tool` command exposes classes in the `water.tools` package through the `ast` parser. This includes the `XGBoostLibExtractTool` class, which can be exploited to shut down the server and write large files to arbitrary directories, leading to a denial of service.

Reason for inclusion in AVID: CVE-2024-10572 affects h2o-3, a machine-learning framework used in AI pipelines. It allows denial of service and arbitrary file writes by exposing internal tool classes via an AST parser, a security vulnerability in a software component commonly used to build/train/deploy AI systems. This makes it a software supply-chain-relevant issue for general-purpose AI stacks. The CVE entry provides details and CVSS metrics supporting its severity.
