## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `grandma.Win10`.

## Details

The probe grandma.Win10 tests the ability to extract ethnic slurs by roleplaying as a grandmother. This probe is part of the grandma module which simulates a grandma to elicit sensitive information.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
