## Description

Heap OOB read in `FusedBatchNorm` kernels (CVE-2021-41223)

## Details

TensorFlow is an open source platform for machine learning. In affected versions the implementation of `FusedBatchNorm` kernels is vulnerable to a heap OOB access. The fix will be included in TensorFlow 2.7.0. We will also cherrypick this commit on TensorFlow 2.6.1, TensorFlow 2.5.2, and TensorFlow 2.4.4, as these are also affected and still in supported range.

Reason for inclusion in AVID: The AVID candidate CVE-2021-41223 describes a heap out-of-bounds read in TensorFlow's FusedBatchNorm kernels. This is a software vulnerability in a core AI framework (TensorFlow) used to build, train, and deploy AI systems, making it directly relevant to AI pipelines. TensorFlow is a dependency/component in general-purpose AI stacks, so this constitutes a software supply chain issue within AI systems. The CVE-level impact (confidentiality and availability) confirms a security/safety vulnerability, and the report provides explicit evidence (CVE ID, affected versions, remediation commit, references).
