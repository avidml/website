## Description

Document Intelligence Studio On-Prem Elevation of Privilege Vulnerability (CVE-2025-30387)

## Details

Improper limitation of a pathname to a restricted directory ('path traversal') in Azure allows an unauthorized attacker to elevate privileges over a network.

Reason for inclusion in AVID: CVE-2025-30387 describes a path traversal vulnerability in Document Intelligence Studio On-Prem (an AI document processing product). It is a software vulnerability in an AI-related component used in AI workflows, with privilege escalation risk. This aligns with AI supply-chain concerns (software used to build/deploy/run AI systems) and is clearly a security vulnerability with high impact signals.
