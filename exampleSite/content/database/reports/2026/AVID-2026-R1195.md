## Description

TensorFlow vulnerable to integer overflow in EditDistance (CVE-2023-25662)

## Details

TensorFlow is an open source platform for machine learning. Versions prior to 2.12.0 and 2.11.1 are vulnerable to integer overflow in EditDistance. A fix is included in TensorFlow version 2.12.0 and version 2.11.1.

Reason for inclusion in AVID: CVE-2023-25662 describes an integer overflow vulnerability in TensorFlow's EditDistance, affecting TensorFlow (an ML framework). This is a software supply chain vulnerability in a component used to build/train/deploy AI systems, with explicit CVE details and a fixed version. It is a security vulnerability with potential availability impact per CVSS, and there is sufficient evidence in the report to classify it for AVID curation.
