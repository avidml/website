## Description

Improper Restriction of XML External Entity Reference in stanfordnlp/corenlp (CVE-2021-3878)

## Details

corenlp is vulnerable to Improper Restriction of XML External Entity Reference

Reason for inclusion in AVID: CVE-2021-3878 describes an XML External Entity (XXE) vulnerability in stanfordnlp/corenlp. CoreNLP is a widely used NLP library in AI pipelines, so this directly concerns AI software, models, and tooling. The issue affects a software component used to build/run AI systems (dependencies in NLP stacks), representing a software supply chain vulnerability. It is a security vulnerability with network-exploitable characteristics (RCE-like impact via XXE) and has clear CVSS evidence. Therefore, this should be kept for AVID curation as a vulnerability in the general-purpose AI software supply chain.
