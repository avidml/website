## Description

Command injection in _wget_download (CVE-2023-52311)

## Details

PaddlePaddle before 2.6.0 has a command injection in _wget_download. This resulted in the ability to execute arbitrary commands on the operating system.

Reason for inclusion in AVID: CVE-2023-52311 describes a command injection (RCE) vulnerability in PaddlePaddle's _wget_download, affecting PaddlePaddle before 2.6.0. PaddlePaddle is a widely used AI framework in ML model training, deployment, and serving pipelines. This is a software vulnerability in a component (framework) that is part of the AI software stack, representing a security flaw within the general-purpose AI supply chain. The CVSS score is 9.6 (CRITICAL) with remote code execution potential, indicating a security/safety risk. Evidence provided (CVE entry and advisory) supports classification as an AI-related, GP AI supply chain vulnerability.
