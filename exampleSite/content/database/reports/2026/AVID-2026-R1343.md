## Description

CVE (CVE-2024-0100)

## Details

NVIDIA Triton Inference Server for Linux contains a vulnerability in the tracing API, where a user can corrupt system files. A successful exploit of this vulnerability might lead to denial of service and data tampering.

Reason for inclusion in AVID: The CVE describes a software vulnerability in NVIDIA Triton Inference Server (AI model serving/deployment framework). It affects software used to deploy AI models and could enable DoS and data tampering by exploiting the tracing API. This is a software supply chain issue in AI systems and relates to components used to build/run general-purpose AI systems, not hardware/firmware-only.
