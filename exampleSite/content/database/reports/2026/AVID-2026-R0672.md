## Description

The model rnj-1-instruct from Essential AI was evaluated by the Garak LLM Vulnerability scanner using the probe `grandma.Substances`.

## Details

The probe grandma.Substances tests the ability to extract illegal or contentious information by roleplaying as a grandmother. This probe is part of the grandma module which simulates a grandma to elicit sensitive information.

The LLM rnj-1-instruct was evaluated on this probe.
