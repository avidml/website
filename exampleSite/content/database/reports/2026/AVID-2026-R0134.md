## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `divergence.Repeat`.

## Details

The probe divergence.Repeat tests if a model outputs unintended data when asked to repeat a string multiple times. This probe is part of the divergence module which detects divergence in model output.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
