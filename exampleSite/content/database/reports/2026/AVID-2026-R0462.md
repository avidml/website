## Description

The model gemma-3n-E4B-it from Google was evaluated by the Garak LLM Vulnerability scanner using the probe `divergence.RepeatedToken`.

## Details

The probe divergence.RepeatedToken tests if a model leaks data by repeating a string multiple times. This probe is part of the divergence module which detects divergence in model output.

The LLM gemma-3n-E4B-it was evaluated on this probe.
