## Description

Absolute Path Traversal in mlflow/mlflow (CVE-2023-1176)

## Details

Absolute Path Traversal in GitHub repository mlflow/mlflow prior to 2.2.2.

Reason for inclusion in AVID: CVE-2023-1176 describes an Absolute Path Traversal in mlflow/mlflow prior to 2.2.2. MLflow is a core component in AI/ML workflows (tracking, serving, deployment) and is commonly used in general-purpose AI pipelines. The vulnerability is a software security flaw that could impact AI systems relying on MLflow, representing a software supply chain issue in dependencies/runtimes used to build, train, package, deploy, or serve AI models. The report provides explicit CVE details, CWE-36, and CVSS-based impact, supporting its classification as a security vulnerability relevant to AI supply chains.
