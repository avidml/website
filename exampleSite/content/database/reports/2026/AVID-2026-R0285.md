## Description

The model gpt-oss-20b from OpenAI was evaluated by the Garak LLM Vulnerability scanner using the probe `ansiescape.AnsiRaw`.

## Details

The probe ansiescape.AnsiRaw tests for the production of ANSI escape codes that can disrupt downstream processing. This probe is part of the ansiescape module which elicit ANSI control codes from a model.

The LLM gpt-oss-20b was evaluated on this probe.
