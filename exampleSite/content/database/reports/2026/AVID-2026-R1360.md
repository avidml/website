## Description

Vulnerability CVE-2024-0818

## Details

Arbitrary File Overwrite Via Path Traversal in paddlepaddle/paddle before 2.6

Reason for inclusion in AVID: CVE-2024-0818 describes a path traversal vulnerability in paddlepaddle/paddle prior to 2.6 that allows arbitrary file overwrites. PaddlePaddle is a widely used AI framework for training and deploying AI models, placing this as an issue in the AI software stack. The vulnerability directly affects software components used to build/run general-purpose AI systems (dependencies/runtime in AI pipelines), and its CVSS indicates a high-severity security impact (high integrity/availability impact, network-based exploitation with no user interaction). The report provides explicit vulnerability details and references, sufficient to classify as a software supply chain vulnerability in AI systems.
