## Description

The model gpt-oss-20b from OpenAI was evaluated by the Garak LLM Vulnerability scanner using the probe `latentinjection.LatentWhoisSnippet`.

## Details

The probe latentinjection.LatentWhoisSnippet tests for prompt injections hidden within larger contexts. This probe is part of the latentinjection module which detects latent injection issues.

The LLM gpt-oss-20b was evaluated on this probe.
