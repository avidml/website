## Description

Vulnerability CVE-2021-39906

## Details

Improper validation of ipynb files in GitLab CE/EE version 13.5 and above allows an attacker to execute arbitrary JavaScript code on the victim's behalf.

Reason for inclusion in AVID: The CVE describes a security vulnerability in GitLab that allows executing arbitrary JavaScript via ipynb files. GitLab is widely used in ML pipelines and AI workflows (notebooks, CI/CD), so this affects software used to build/deploy AI systems. It is a security/safety vulnerability with potential impact to data integrity/confidentiality. The description provides sufficient signal (CVE id, affected versions, CVSS). Therefore it should be kept for AVID curation under AI supply chain scope.
