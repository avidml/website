## Description

Vulnerability CVE-2024-0110

## Details

NVIDIA CUDA Toolkit contains a vulnerability in command `cuobjdump` where a user may cause an out-of-bound write by passing in a malformed ELF file. A successful exploit of this vulnerability may lead to code execution or denial of service.

Reason for inclusion in AVID: The CVE describes a software vulnerability in NVIDIA CUDA Toolkit's cuobjdump that can cause out-of-bounds writes, potentially enabling code execution or DoS. CUDA Toolkit is a core software component used in AI model development, deployment, and tooling, making it AI-related and part of the software supply chain for general-purpose AI systems. The issue affects a tool/library within the AI stack (not hardware-only), thus fitting GP AI supply chain concerns. The CVE provides explicit vulnerability behavior and impact signals, supporting its inclusion.
