## Description

chidiwilliams buzz model_loader.py download_model temp file (CVE-2024-10372)

## Details

A vulnerability classified as problematic was found in chidiwilliams buzz 1.1.0. This vulnerability affects the function download_model of the file buzz/model_loader.py. The manipulation leads to insecure temporary file. It is possible to launch the attack on the local host. The complexity of an attack is rather high. The exploitation appears to be difficult. The exploit has been disclosed to the public and may be used. The vendor was contacted early about this disclosure but did not respond in any way.

Reason for inclusion in AVID: The CVE concerns an AI-related library (buzz) used to download/load models (model_loader.py download_model). The insecure temporary file vulnerability is a software risk in the AI stack (model download/loading) that could affect model integrity or data on systems running AI workloads. It targets a dependency used in ML pipelines, thus a software supply-chain issue for general-purpose AI systems. Hardware/firmware-only issues are not implicated, and the report provides CVE, CWE, and references.
