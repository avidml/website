## Description

Path Traversal in mlflow/mlflow (CVE-2023-6753)

## Details

Path Traversal in GitHub repository mlflow/mlflow prior to 2.9.2.

Reason for inclusion in AVID: The CVE reports a path traversal vulnerability in MLflow (an AI/ML tooling component) affecting mlflow/mlflow before 2.9.2. This is a software vulnerability that impacts AI pipelines/tools used to build/train/deploy AI systems; it resides in a software component (MLflow) that is part of AI supply chains. The vulnerability has CVSS high severity (network-based with user interaction and high impacts). Sufficient evidence is provided (CVE entry and references).
