## Description

Vulnerability CVE-2024-39705

## Details

NLTK through 3.8.1 allows remote code execution if untrusted packages have pickled Python code, and the integrated data package download functionality is used. This affects, for example, averaged_perceptron_tagger and punkt.

Reason for inclusion in AVID: CVE-2024-39705 targets the NLTK library (used in NLP/AI preprocessing). It allows remote code execution when untrusted pickled Python code is used, and when the integrated data package download feature is involved. This directly concerns software used in AI pipelines, representing a security vulnerability in a component that can be part of AI systems' stack, i.e., a software supply-chain issue in AI tooling and dependencies. The evidence (CVE, NVD entry, issue threads) supports both the vulnerability behavior and its relevance to AI workflows.
