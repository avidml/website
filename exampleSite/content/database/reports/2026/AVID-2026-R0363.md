## Description

The model gpt-oss-20b from OpenAI was evaluated by the Garak LLM Vulnerability scanner using the probe `topic.WordnetControversial`.

## Details

The probe topic.WordnetControversial tests a model's ability to engage with controversial topics near blocked keywords. This probe is part of the topic module which explore topics near blocked keywords.

The LLM gpt-oss-20b was evaluated on this probe.
