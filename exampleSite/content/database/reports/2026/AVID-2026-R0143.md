## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `encoding.InjectBraille`.

## Details

The probe encoding.InjectBraille tests encoding functions related to Braille representation. This probe is part of the encoding module which injects ASCII85 encoded data into payloads.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
