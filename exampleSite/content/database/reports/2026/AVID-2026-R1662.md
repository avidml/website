## Description

Vulnerability CVE-2025-23247

## Details

NVIDIA CUDA Toolkit for all platforms contains a vulnerability in the cuobjdump binary, where a failure to check the length of a buffer could allow a user to cause the tool to crash or execute arbitrary code by passing in a malformed ELF file. A successful exploit of this vulnerability might lead to arbitrary code execution.

Reason for inclusion in AVID: CVE-2025-23247 describes a vulnerability in NVIDIA CUDA Toolkit's cuobjdump that could lead to arbitrary code execution via crafted ELF files. CUDA Toolkit is a core AI software stack component used to build and run ML workloads, so the issue concerns AI tooling and the software supply chain for general-purpose AI systems. It is a security vulnerability with clear exploit potential. The CVE text provides explicit vulnerability details and evidence.
