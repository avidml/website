## Description

Deserialization of Untrusted Data in huggingface/transformers (CVE-2023-7018)

## Details

Deserialization of Untrusted Data in GitHub repository huggingface/transformers prior to 4.36.

Reason for inclusion in AVID: CVE-2023-7018 describes a deserialization vulnerability in the huggingface/transformers library (pre-4.36). This is a security flaw in a widely used AI software library that is integral to model development, deployment, and inference pipelines. It directly concerns software used to build and run general-purpose AI systems, and if exploited could lead to RCE or data integrity issues. Therefore it is a relevant software supply-chain vulnerability for AI systems.
