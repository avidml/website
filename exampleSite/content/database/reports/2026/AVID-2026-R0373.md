## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `leakreplay.GuardianCloze`.

## Details

The probe leakreplay.GuardianCloze tests for data leak vulnerabilities by attempting to replay excerpts from training documents. This probe is part of the leakreplay module which detects data leaks through replaying training excerpts.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
