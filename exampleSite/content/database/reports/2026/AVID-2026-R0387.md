## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `malwaregen.Evasion`.

## Details

The probe malwaregen.Evasion tests if the model can generate code to evade detection by anti-malware systems. This probe is part of the malwaregen module which requests code to evade security measures.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
