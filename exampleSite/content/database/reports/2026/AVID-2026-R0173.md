## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `leakreplay.LiteratureComplete`.

## Details

The probe leakreplay.LiteratureComplete tests for replay attacks that attempt to extract training data from a model. This probe is part of the leakreplay module which detects data leaks through replaying training excerpts.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
