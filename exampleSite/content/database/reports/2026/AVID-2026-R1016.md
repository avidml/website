## Description

Missing validation causes `TensorSummaryV2` in TensorFlow to crash (CVE-2022-29193)

## Details

TensorFlow is an open source platform for machine learning. Prior to versions 2.9.0, 2.8.1, 2.7.2, and 2.6.4, the implementation of `tf.raw_ops.TensorSummaryV2` does not fully validate the input arguments. This results in a `CHECK`-failure which can be used to trigger a denial of service attack. Versions 2.9.0, 2.8.1, 2.7.2, and 2.6.4 contain a patch for this issue.

Reason for inclusion in AVID: This CVE targets TensorFlow's TensorSummaryV2 input validation, causing a denial-of-service via a CHECK failure. It is a software vulnerability in a widely used ML framework, not hardware/firmware-only. TensorFlow is a key component in AI software stacks, so this affects the software supply chain for AI systems. The report cites affected versions, patches, and CVSS details, providing sufficient evidence for classification.
