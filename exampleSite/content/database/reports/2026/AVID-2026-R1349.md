## Description

Vulnerability CVE-2024-0116

## Details

NVIDIA Triton Inference Server contains a vulnerability where a user may cause an out-of-bounds read issue by releasing a shared memory region while it is in use. A successful exploit of this vulnerability may lead to denial of service.

Reason for inclusion in AVID: CVE-2024-0116 describes a software vulnerability in NVIDIA Triton Inference Server (AI serving stack) that allows an out-of-bounds read leading to denial of service. This affects a component used to build/run general-purpose AI systems, is a software supply-chain-relevant issue, and is a clearly defined security vulnerability (CWE-125) with CVSS impact. The report provides sufficient signals (description, CVSS, CWE, references) for classification.
