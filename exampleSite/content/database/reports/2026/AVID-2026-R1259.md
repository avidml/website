## Description

Vulnerability CVE-2023-38669

## Details

Use after free in paddle.diagonal in PaddlePaddle before 2.5.0. This resulted in a potentially exploitable condition.

Reason for inclusion in AVID: CVE-2023-38669 describes a use-after-free vulnerability in PaddlePaddle (paddle.diagonal) prior to 2.5.0. PaddlePaddle is a widely used AI framework, and this vulnerability affects software components commonly used to build, train, and deploy general-purpose AI systems. It is a clear security vulnerability (CWE-416) with public CVE/NVD references, signaling potential exploitation within AI software stacks. This aligns with software supply-chain risk in AI pipelines through vulnerable dependencies or runtimes.
