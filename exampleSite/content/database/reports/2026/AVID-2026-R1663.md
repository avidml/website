## Description

Vulnerability CVE-2025-23249

## Details

NVIDIA NeMo Framework contains a vulnerability where a user could cause a deserialization of untrusted data by remote code execution. A successful exploit of this vulnerability might lead to code execution and data tampering.

Reason for inclusion in AVID: The CVE describes a deserialization vulnerability in NVIDIA NeMo Framework enabling remote code execution, a known security risk. NeMo is an AI framework used in ML pipelines, modeling, and deployment, so this is a software supply-chain relevant issue affecting components used to build/run general-purpose AI systems. The vulnerability is clearly security-related with RCE/tampering, and the report provides sufficient evidence (CVE, description, CVSS, references).
