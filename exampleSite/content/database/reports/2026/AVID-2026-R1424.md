## Description

Spring Cloud Function Web DOS Vulnerability (CVE-2024-22271)

## Details

In Spring Cloud Function framework, versions 4.1.x prior to 4.1.2, 4.0.x prior to 4.0.8 an application is vulnerable to a DOS attack when attempting to compose functions with non-existing functions.

Specifically, an application is vulnerable when all of the following are true:

User is using Spring Cloud Function Web module

Affected Spring Products and Versions Spring Cloud Function Framework 4.1.0 to 4.1.2 4.0.0 to 4.0.8

References  https://spring.io/security/cve-2022-22979   https://checkmarx.com/blog/spring-function-cloud-dos-cve-2022-22979-and-unintended-function-invocation/  History 2020-01-16: Initial vulnerability report published.

Reason for inclusion in AVID: CVE-2024-22271 describes a DoS vulnerability in the Spring Cloud Function Web module that affects framework versions 4.1.x and 4.0.x. This is a software vulnerability in a dependency that could be used to build or deploy AI-serving microservices, making it relevant to the software supply chain of general-purpose AI systems. The issue concerns a vulnerability behavior (DoS) with high severity (CVSS 8.2) and is documented with affected versions and references, satisfying evidence requirements. It is not hardware/firmware-only. Therefore, it should be kept for AVID curation as a vulnerability in the AI supply chain.
