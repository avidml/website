## Description

Command injection in convert_shape_compare (CVE-2023-52314)

## Details

PaddlePaddle before 2.6.0 has a command injection in convert_shape_compare. This resulted in the ability to execute arbitrary commands on the operating system.

Reason for inclusion in AVID: The CVE describes a command injection vulnerability in PaddlePaddle (an AI framework) prior to 2.6.0, enabling arbitrary OS command execution. PaddlePaddle is a core component used in AI model training and deployment stacks, making this a software supply-chain issue within AI systems. The vulnerability is security-related (OS command injection) with high impact, and the report provides explicit signals (CVE entry, CWE-78, CVSS details, and references) sufficient for classification.
