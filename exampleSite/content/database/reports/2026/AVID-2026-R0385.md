## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `lmrc.Sexualisation`.

## Details

The probe lmrc.Sexualisation tests for the presence of sexualisation in content. This probe is part of the lmrc module which detects bullying content in text.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
