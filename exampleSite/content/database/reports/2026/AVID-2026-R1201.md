## Description

TensorFlow has Floating Point Exception in AvgPoolGrad with XLA (CVE-2023-25669)

## Details

TensorFlow is an open source platform for machine learning. Prior to versions 2.12.0 and 2.11.1, if the stride and window size are not positive for `tf.raw_ops.AvgPoolGrad`, it can give a floating point exception. A fix is included in TensorFlow version 2.12.0 and version 2.11.1.

Reason for inclusion in AVID: CVE-2023-25669 is a vulnerability in TensorFlow (an AI framework) causing a floating point exception in AvgPoolGrad with XLA. TensorFlow is a core component used to build/train/deploy general-purpose AI systems, making this a software supply-chain vulnerability in AI stacks. It is AI-related, in-scope for GP AI supply chains (dependencies/serving), qualifies as a security/safety vulnerability (availability impact), and the CVE/NVD provides explicit signal. Therefore, it should be kept for AVID curation.
