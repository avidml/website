## Description

Vulnerability CVE-2025-23243

## Details

NVIDIA Riva contains a vulnerability where a user could cause an improper access control issue. A successful exploit of this vulnerability might lead to data tampering or denial of service.

Reason for inclusion in AVID: CVE-2025-23243 affects NVIDIA Riva, an AI inference/serving component. It describes an improper access control vulnerability that could lead to data tampering or denial of service. Riva is a software component used in AI pipelines and deployments, making this a software supply-chain vulnerability within general-purpose AI systems. The CVE entry, CWE reference, and CVSS details provide sufficient evidence.
