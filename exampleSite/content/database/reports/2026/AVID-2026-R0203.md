## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `suffix.GCGCached`.

## Details

The probe suffix.GCGCached tests the effectiveness of adversarial suffix attacks on system prompts. This probe is part of the suffix module which append adversarial suffixes to system prompts.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
