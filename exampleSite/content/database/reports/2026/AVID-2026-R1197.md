## Description

TensorFlow vulnerable to Heap Buffer Overflow in AvgPoolGrad (CVE-2023-25664)

## Details

TensorFlow is an open source platform for machine learning. Prior to versions 2.12.0 and 2.11.1, there is a heap buffer overflow in TAvgPoolGrad. A fix is included in TensorFlow 2.12.0 and 2.11.1.

Reason for inclusion in AVID: CVE-2023-25664 describes a heap-based buffer overflow in TensorFlow's AvgPoolGrad, a software vulnerability in a widely-used ML framework. Affects TensorFlow versions prior to 2.12.0 and 2.11.1; mitigated by updates. TensorFlow is a core AI software stack used to build/train/deploy AI systems, so this is a software supply-chain issue in AI tooling. The CVSS/description indicate a security-related vulnerability with potential denial-of-service via memory corruption. Sufficient public signal is provided (CVE entry, advisory, commit).
