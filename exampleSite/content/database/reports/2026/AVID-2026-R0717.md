## Description

Vulnerability CVE-2021-22205

## Details

An issue has been discovered in GitLab CE/EE affecting all versions starting from 11.9. GitLab was not properly validating image files that were passed to a file parser which resulted in a remote command execution.

Reason for inclusion in AVID: CVE-2021-22205 describes an unauthenticated remote code execution vulnerability in GitLab CE/EE via improper validation of image files passed to a parser. GitLab is widely used in AI/ML pipelines for build/test/deploy and artifact management, so a vulnerability in this component constitutes a software supply-chain risk for general-purpose AI systems. The issue is clearly a security vulnerability with high impact, and the report provides concrete details and CVSS metrics.
