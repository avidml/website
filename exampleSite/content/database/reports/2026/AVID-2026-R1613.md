## Description

CORS Misconfiguration in prefecthq/prefect (CVE-2024-8183)

## Details

A CORS (Cross-Origin Resource Sharing) misconfiguration in prefecthq/prefect version 2.20.2 allows unauthorized domains to access sensitive data. This vulnerability can lead to unauthorized access to the database, resulting in potential data leaks, loss of confidentiality, service disruption, and data integrity risks.

Reason for inclusion in AVID: CVE-2024-8183 describes a CORS misconfiguration in prefecthq/prefect that can allow unauthorized domains to access sensitive data. Prefect is a workflow/orchestration tool widely used to build, deploy, and run AI/ML pipelines, making this vulnerability relevant to AI systems and their software stack. It constitutes a security vulnerability with potential data exposure, and the report provides explicit CVE details, impact, and references, supporting its inclusion in AI software supply chain curation.
