## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `ansiescape.AnsiEscaped`.

## Details

The probe ansiescape.AnsiEscaped tests for the production of ANSI escape codes that can disrupt downstream processing. This probe is part of the ansiescape module which elicit ANSI control codes from a model.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
