## Description

Server-Side Request Forgery (SSRF) in kubeflow/kubeflow (CVE-2023-6570)

## Details

Server-Side Request Forgery (SSRF) in kubeflow/kubeflow

Reason for inclusion in AVID: The CVE describes a server-side request forgery (SSRF) in Kubeflow, which is an ML workflow/orchestration platform widely used in AI pipelines. This directly concerns components used to build, train, deploy, and run AI systems, i.e., the software supply chain of general-purpose AI systems. It is a security vulnerability with clear CVE/SSRF details and references.
