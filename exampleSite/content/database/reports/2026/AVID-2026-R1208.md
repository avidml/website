## Description

TensorFlow has null dereference on ParallelConcat with XLA (CVE-2023-25676)

## Details

TensorFlow is an open source machine learning platform. When running versions prior to 2.12.0 and 2.11.1 with XLA, `tf.raw_ops.ParallelConcat` segfaults with a nullptr dereference when given a parameter `shape` with rank that is not greater than zero. A fix is available in TensorFlow 2.12.0 and 2.11.1.

Reason for inclusion in AVID: CVE-2023-25676 describes a NULL pointer dereference vulnerability in TensorFlow when using XLA with ParallelConcat. TensorFlow is a core AI framework used to build/train/deploy AI systems, so this is a software supply chain issue within the AI software stack. The CVE provides affected versions, fixes, and CVSS details, giving sufficient evidence that it is a security vulnerability impacting AI pipelines.
