## Description

Vulnerability CVE-2022-0668

## Details

JFrog Artifactory prior to 7.37.13 is vulnerable to Authentication Bypass, which can lead to Privilege Escalation when a specially crafted request is sent by an unauthenticated user.

Reason for inclusion in AVID: CVE-2022-0668 describes an authentication bypass in JFrog Artifactory, enabling privilege escalation. Artifactory is a software artifact repository commonly used in AI/ML development pipelines to store dependencies, models, and build artifacts. A vulnerability in this component can compromise the software supply chain underpinning AI systems (CI/CD, artifact management, model artifacts). Therefore it is a software supply chain issue relevant to general-purpose AI systems and constitutes a security vulnerability with clear evidence.
