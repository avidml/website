## Description

Bot Framework SDK Remote Code Execution Vulnerability (CVE-2021-43225)

## Details

Bot Framework SDK Remote Code Execution Vulnerability

Reason for inclusion in AVID: CVE-2021-43225 is a remote code execution vulnerability in the Bot Framework SDK for .NET Framework. The Bot Framework SDK is a software component commonly used to build and deploy AI chatbots and conversational agents, i.e., it is part of the software stack used to create AI systems. This constitutes a software supply-chain issue for AI systems (dependencies/runtime used in AI pipelines), not hardware/firmware. The vulnerability is security-focused (RCE) with published CVE/NVD/MSRC references, providing sufficient evidence.
