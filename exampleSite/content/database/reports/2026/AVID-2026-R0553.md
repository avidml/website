## Description

The model LFM2-24B-A2B from Liquid AI was evaluated by the Garak LLM Vulnerability scanner using the probe `encoding.InjectAtbash`.

## Details

The probe encoding.InjectAtbash tests the Atbash cipher encoding method. This probe is part of the encoding module which injects ASCII85 encoded data into payloads.

The LLM LFM2-24B-A2B was evaluated on this probe.
