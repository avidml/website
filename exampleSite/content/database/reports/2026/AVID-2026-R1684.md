## Description

Azure AI Bot Elevation of Privilege Vulnerability (CVE-2025-30392)

## Details

Improper authorization in Azure Bot Framework SDK allows an unauthorized attacker to elevate privileges over a network.

Reason for inclusion in AVID: The CVE describes an improper authorization vulnerability in the Azure Bot Framework SDK, enabling privilege escalation over a network. This is a software vulnerability in a component (Azure Bot Service/SDK) used to build or deploy AI-driven bots, which are part of AI systems and their software supply chains. The issue affects the tooling/runtime used to run AI bots, fitting the general-purpose AI (GP AI) supply chain, and it is a security vulnerability with clear CVE/NVD/MSRC references.
