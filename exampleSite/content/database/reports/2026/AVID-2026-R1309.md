## Description

FPE in paddle.lerp (CVE-2023-52306)

## Details

FPE in paddle.lerp in PaddlePaddle before 2.6.0. This flaw can cause a runtime crash and a denial of service.

Reason for inclusion in AVID: CVE-2023-52306 describes a runtime crash/denial-of-service due to a divide-by-zero in PaddlePaddle's paddle.lerp. PaddlePaddle is a widely-used AI framework; a vulnerability in this software component is directly relevant to AI systems that rely on PaddlePaddle for training, inference, or pipeline orchestration. This is a software supply-chain vulnerability in a dependency/framework used to build/run general-purpose AI systems, not hardware/firmware-only. It is a security/safety vulnerability with potential availability impact, and explicit CVE evidence is provided.
