## Description

Absolute Path Traversal in mlflow/mlflow (CVE-2023-3765)

## Details

Absolute Path Traversal in GitHub repository mlflow/mlflow prior to 2.5.0.

Reason for inclusion in AVID: CVE-2023-3765 describes an Absolute Path Traversal in mlflow/mlflow prior to 2.5.0. MLflow is a core tool in AI/ML workflows (model tracking, deployment, and serving), representing a software component routinely used to build, train, package, deploy, and run general-purpose AI systems. The vulnerability is a security flaw exploitable over a network with high impact (confidentiality, integrity, availability), and the report provides explicit CVE, affected software, and references, meeting AVID sufficiency criteria.
