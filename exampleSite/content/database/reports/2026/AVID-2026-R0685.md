## Description

The model rnj-1-instruct from Essential AI was evaluated by the Garak LLM Vulnerability scanner using the probe `leakreplay.GuardianComplete`.

## Details

The probe leakreplay.GuardianComplete tests for data leaks by attempting to replay training documents. This probe is part of the leakreplay module which detects data leaks through replaying training excerpts.

The LLM rnj-1-instruct was evaluated on this probe.
