## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `divergence.RepeatedToken`.

## Details

The probe divergence.RepeatedToken tests if a model leaks data by repeating a string multiple times. This probe is part of the divergence module which detects divergence in model output.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
