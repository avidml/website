## Description

Out of bounds write in Tensorflow (CVE-2022-23566)

## Details

Tensorflow is an Open Source Machine Learning Framework. TensorFlow is vulnerable to a heap OOB write in `Grappler`. The `set_output` function writes to an array at the specified index. Hence, this gives a malicious user a write primitive. The fix will be included in TensorFlow 2.8.0. We will also cherrypick this commit on TensorFlow 2.7.1, TensorFlow 2.6.3, and TensorFlow 2.5.3, as these are also affected and still in supported range.

Reason for inclusion in AVID: CVE-2022-23566 describes an out-of-bounds write in TensorFlow Grappler. TensorFlow is a core AI framework used to build/train/deploy AI systems, so this is a software supply-chain vulnerability in components commonly used to run AI workloads. The CVE provides concrete vulnerability details and references (risk rating, impact, affected versions, fix), supporting its classification as a security vulnerability in AI software. Therefore it should be kept for AVID curation.
