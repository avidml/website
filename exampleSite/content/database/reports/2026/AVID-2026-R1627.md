## Description

h2oai h2o-3 JDBC Connection 1 getConnectionSafe deserialization (CVE-2024-8862)

## Details

A vulnerability, which was classified as critical, has been found in h2oai h2o-3 3.46.0.4. This issue affects the function getConnectionSafe of the file /dtale/chart-data/1 of the component JDBC Connection Handler. The manipulation of the argument query leads to deserialization. The attack may be initiated remotely. The exploit has been disclosed to the public and may be used. The vendor was contacted early about this disclosure but did not respond in any way.

Reason for inclusion in AVID: CVE-2024-8862 describes a deserialization vulnerability in the h2o-3 JDBC Connection Handler, a component used in AI/ML data processing pipelines. This is a vulnerability in a software dependency relevant to building, training, deploying, or running general-purpose AI systems, i.e., a software supply-chain issue in AI stacks. It enables remote execution (security/safety risk) and is backed by CVE, CWE-502, and public disclosure details, providing sufficient evidence for AVID curation.
