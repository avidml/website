## Description

Null pointer dereference in paddle.flip (CVE-2023-38670)

## Details

Null pointer dereference in paddle.flip in PaddlePaddle before 2.5.0. This resulted in a runtime crash and denial of service.

Reason for inclusion in AVID: CVE-2023-38670 describes a null pointer dereference in PaddlePaddle's paddle.flip, causing runtime crashes and potential denial of service. PaddlePaddle is a major ML/DL framework used to build, train, and deploy AI systems; a vulnerability in this component constitutes a software supply-chain issue for general-purpose AI systems. The report includes CVE details, CVSS metrics, and references, providing sufficient signal to classify as a security risk in the AI software supply chain.
