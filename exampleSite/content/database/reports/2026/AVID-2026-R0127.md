## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `apikey.CompleteKey`.

## Details

The probe apikey.CompleteKey tests if the system can complete partial API keys for various services. This probe is part of the apikey module which prompts to generate complete API keys.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
