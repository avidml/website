## Description

IBM Watson Studio Local cross-site request forgery (CVE-2024-49340)

## Details

IBM Watson Studio Local 1.2.3 is vulnerable to cross-site request forgery which could allow an attacker to execute malicious and unauthorized actions transmitted from a user that the website trusts.

Reason for inclusion in AVID: CVE-2024-49340 is a CSRF vulnerability affecting IBM Watson Studio Local, an AI development/production tool used in ML pipelines. It is a software vulnerability in a component used to build/run AI systems, i.e., part of the general-purpose AI software supply chain. The issue concerns security (CSRF), with evidence in the description and references.
