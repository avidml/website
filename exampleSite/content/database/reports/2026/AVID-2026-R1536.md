## Description

Vulnerability CVE-2024-45857

## Details

Deserialization of untrusted data can occur in versions 2.4.0 or newer of the Cleanlab project, enabling a maliciously crafted datalab.pkl file to run arbitrary code on an end user’s system when the data directory is loaded.

Reason for inclusion in AVID: CVE-2024-45857 describes a deserialization vulnerability in the Cleanlab library (versions >= 2.4.0) that allows arbitrary code execution when loading a crafted datalab.pkl. Cleanlab is an ML data preprocessing/cleaning library commonly used in AI pipelines, making this a software supply-chain vulnerability in AI systems. It affects software components used to build/run AI systems. The vulnerability is clearly security-related (RCE). Evidence provided includes CVE entry and description.
