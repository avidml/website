## Description

Denial of Service in h2oai/h2o-3 (CVE-2024-8062)

## Details

A vulnerability in the typeahead endpoint of h2oai/h2o-3 version 3.46.0 allows for a denial of service. The endpoint performs a `HEAD` request to verify the existence of a specified resource without setting a timeout. An attacker can exploit this by sending multiple requests to an attacker-controlled server that hangs, causing the application to block and become unresponsive to other requests.

Reason for inclusion in AVID: CVE-2024-8062 describes a Denial of Service vulnerability in the h2oai/h2o-3 machine learning framework. This vulnerability affects a software component commonly used in AI model building/serving pipelines, impacting availability of AI services. It is a vulnerability in a software package (not hardware/firmware) that is part of AI stacks, thus relevant to the general-purpose AI systems supply chain. The report provides CVE text, description, impact (including CVSS 3.0 metrics), and references, supplying sufficient evidence for classification.
