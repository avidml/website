## Description

Vulnerability CVE-2023-25513

## Details

NVIDIA CUDA toolkit for Linux and Windows contains a vulnerability in cuobjdump, where an attacker may cause an out-of-bounds read by tricking a user into running cuobjdump on a malformed input file. A successful exploit of this vulnerability may lead to limited denial of service, code execution, and limited information disclosure.

Reason for inclusion in AVID: CVE-2023-25513 describes a local out-of-bounds read in cuobjdump within the NVIDIA CUDA toolkit, potentially leading to code execution, information disclosure, or DoS. cuobjdump is part of the CUDA toolkit, a software component widely used in AI workflows (GPUs, ML frameworks, tooling, and deployment/ocean stacks). As such, it is a software supply-chain component affecting general-purpose AI systems. The report provides explicit vulnerability behavior and impacts, satisfying evidence requirements.
