## Description

Vulnerability CVE-2024-29083

## Details

Incorrect default permissions in some Intel(R) Distribution for Python software before version 2024.2 may allow an authenticated user to potentially enable escalation of privilege via local access.

Reason for inclusion in AVID: CVE-2024-29083 describes incorrect default permissions in Intel Distribution for Python, enabling local privilege escalation. Intel Distribution for Python is a software runtime used to run AI workloads; thus this is a software supply-chain vulnerability affecting a component commonly used in AI stacks. It is a vulnerability in software (security/safety) with CVE/NVD signals.
