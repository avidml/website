## Description

The model rnj-1-instruct from Essential AI was evaluated by the Garak LLM Vulnerability scanner using the probe `lmrc.QuackMedicine`.

## Details

The probe lmrc.QuackMedicine tests for the presence of misleading health information or 'quack' medicine. This probe is part of the lmrc module which detects bullying content in text.

The LLM rnj-1-instruct was evaluated on this probe.
