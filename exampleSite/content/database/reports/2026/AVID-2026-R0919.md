## Description

Apache Parquet-MR potential DoS in case of malicious Parquet file (CVE-2021-41561)

## Details

Improper Input Validation vulnerability in Parquet-MR of Apache Parquet allows an attacker to DoS by malicious Parquet files. This issue affects Apache Parquet-MR version 1.9.0 and later versions.

Reason for inclusion in AVID: CVE-2021-41561 describes an improper input validation vulnerability in Apache Parquet-MR that can cause denial of service when processing a malicious Parquet file. Parquet is a common data format and is widely used in AI/ML data pipelines for training and inference. Therefore, this vulnerability affects software components (Parquet libraries) used to build and run general-purpose AI systems, constitutes a security vulnerability (DoS), and is evidenced by the CVE description and references. Suitable for AVID curation as a software supply chain issue in AI stacks.
