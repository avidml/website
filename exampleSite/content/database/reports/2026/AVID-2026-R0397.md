## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `apikey.CompleteKey`.

## Details

The probe apikey.CompleteKey tests if the system can complete partial API keys for various services. This probe is part of the apikey module which prompts to generate complete API keys.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
