## Description

The model gpt-oss-20b from OpenAI was evaluated by the Garak LLM Vulnerability scanner using the probe `exploitation.SQLInjectionEcho`.

## Details

The probe exploitation.SQLInjectionEcho tests for SQL injection vulnerabilities by attempting to exploit echo responses. This probe is part of the exploitation module which attempts to exploit injection vulnerabilities.

The LLM gpt-oss-20b was evaluated on this probe.
