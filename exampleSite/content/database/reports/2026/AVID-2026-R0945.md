## Description

Overflow and uncaught divide by zero in Tensorflow (CVE-2022-21729)

## Details

Tensorflow is an Open Source Machine Learning Framework. The implementation of `UnravelIndex` is vulnerable to a division by zero caused by an integer overflow bug. The fix will be included in TensorFlow 2.8.0. We will also cherrypick this commit on TensorFlow 2.7.1, TensorFlow 2.6.3, and TensorFlow 2.5.3, as these are also affected and still in supported range.

Reason for inclusion in AVID: CVE describes a software vulnerability in TensorFlow's UnravelIndex (integer overflow leading to division by zero) affecting a core AI framework. This is a software supply chain issue in a component (TensorFlow) used to build/run AI systems, with potential denial of service/crash (availability) impact. Evidence includes CVE entry, description, and affected versions and fix notes.
