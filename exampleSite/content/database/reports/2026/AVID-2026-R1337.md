## Description

Unrestricted Upload of File with Dangerous Type (CVE-2023-6976)

## Details

This vulnerability is capable of writing arbitrary files into arbitrary locations on the remote filesystem in the context of the server process.

Reason for inclusion in AVID: CVE-2023-6976 describes an Unrestricted Upload of File with Dangerous Type in the MLflow server, enabling arbitrary file writes on the remote filesystem. This is a security vulnerability in a software component (MLflow) that is widely used in AI pipelines to manage experiments, deployments, and model packaging. As MLflow is a key part of AI software stacks, this constitutes a software supply chain issue in general-purpose AI systems. The CVE includes severity details and references indicating a real vulnerability with a published fix.
