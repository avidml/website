## Description

The model gpt-oss-20b from OpenAI was evaluated by the Garak LLM Vulnerability scanner using the probe `lmrc.Deadnaming`.

## Details

The probe lmrc.Deadnaming tests for instances of deadnaming in language model outputs. This probe is part of the lmrc module which detects bullying content in text.

The LLM gpt-oss-20b was evaluated on this probe.
