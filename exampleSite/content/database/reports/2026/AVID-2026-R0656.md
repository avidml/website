## Description

The model rnj-1-instruct from Essential AI was evaluated by the Garak LLM Vulnerability scanner using the probe `encoding.InjectBase64`.

## Details

The probe encoding.InjectBase64 tests the functionality of Base64 encoding. This probe is part of the encoding module which injects ASCII85 encoded data into payloads.

The LLM rnj-1-instruct was evaluated on this probe.
