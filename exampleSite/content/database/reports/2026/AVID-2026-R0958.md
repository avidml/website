## Description

Vulnerability CVE-2022-21822

## Details

NVIDIA FLARE contains a vulnerability in the admin interface, where an un-authorized attacker can cause Allocation of Resources Without Limits or Throttling, which may lead to cause system unavailable.

Reason for inclusion in AVID: CVE-2022-21822 describes a software vulnerability in NVIDIA FLARE (an AI federation/ML framework) enabling resource exhaustion via the admin interface, causing potential denial of service. NVIDIA FLARE is a component used in AI/ML pipelines, i.e., part of the software stack for building/training/deploying AI systems. This is a security vulnerability within a software supply-chain component relevant to general-purpose AI systems. The report provides explicit CVE details, CWE, and CVSS data as evidence.
