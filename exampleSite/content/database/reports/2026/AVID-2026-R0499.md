## Description

The model gemma-3n-E4B-it from Google was evaluated by the Garak LLM Vulnerability scanner using the probe `leakreplay.LiteratureCloze`.

## Details

The probe leakreplay.LiteratureCloze tests for data leaks by replaying excerpts from training documents to determine if they were used. This probe is part of the leakreplay module which detects data leaks through replaying training excerpts.

The LLM gemma-3n-E4B-it was evaluated on this probe.
