## Description

TensorFlow has Null Pointer Error in LookupTableImportV2 (CVE-2023-25672)

## Details

TensorFlow is an open source platform for machine learning. The function `tf.raw_ops.LookupTableImportV2` cannot handle scalars in the `values` parameter and gives an NPE. A fix is included in TensorFlow version 2.12.0 and version 2.11.1.

Reason for inclusion in AVID: CVE-2023-25672 describes a Null Pointer Dereference vulnerability in TensorFlow (LookupTableImportV2). TensorFlow is a core ML framework used in training and deploying AI systems, so this is a software supply chain issue affecting AI software dependencies. The report provides explicit vulnerability details and a fix version, supporting its relevance for AVID curation.
