## Description

Vulnerability CVE-2024-37061

## Details

Remote Code Execution can occur in versions of the MLflow platform running version 1.11.0 or newer, enabling a maliciously crafted MLproject to execute arbitrary code on an end user’s system when run.

Reason for inclusion in AVID: CVE-2024-37061 describes a remote code execution vulnerability in MLflow (versions 1.11.0+), triggered by a maliciously crafted MLproject, enabling arbitrary code execution on the end user’s system. MLflow is a core tool used in AI/ML pipelines for building, training, packaging, deploying, and serving AI models, making this vulnerability relevant to AI systems. The issue affects a software component used to build/run AI systems, which places it in the general-purpose AI software supply chain. It is a security vulnerability (code execution) with explicit CVE details and references, satisfying AVID’s criteria for exposure in AI supply chains.
