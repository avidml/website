## Description

Vulnerability CVE-2023-25512

## Details

NVIDIA CUDA toolkit for Linux and Windows contains a vulnerability in cuobjdump, where an attacker may cause an out-of-bounds memory read by running cuobjdump on a malformed input file. A successful exploit of this vulnerability may lead to limited denial of service, code execution, and limited information disclosure.

Reason for inclusion in AVID: CVE-2023-25512 describes a vulnerability in NVIDIA CUDA Toolkit's cuobjdump that allows an attacker to trigger an out-of-bounds read, potentially enabling code execution or information disclosure. CUDA Toolkit is a core software dependency used to build, train, deploy, and run AI systems (GPUs, libraries, and toolchains). This is a software supply-chain issue within a component commonly used in AI pipelines, not a hardware/firmware-only vulnerability. The CVE indicates plausible exploitation (local, with user interaction) affecting the software stack of AI workloads.
