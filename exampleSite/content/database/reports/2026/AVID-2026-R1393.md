## Description

S3 Bucket Takeover in h2oai/h2o-3 (CVE-2024-1456)

## Details

An S3 bucket takeover vulnerability was identified in the h2oai/h2o-3 repository. The issue involves the S3 bucket 'http://s3.amazonaws.com/h2o-training', which was found to be vulnerable to unauthorized takeover.

Reason for inclusion in AVID: CVE-2024-1456 describes a S3 bucket takeover vulnerability in the h2oai/h2o-3 AI framework, affecting AI model training/deployment workflows. It is a software supply chain issue within AI software stacks (data ingress/storage and related logic). The vulnerability is security-related (unauthorized access, data integrity/confidentiality impact) and the report provides explicit signals (CVSS metrics, CVE reference, external references). Therefore, it should be kept for AVID curation.
