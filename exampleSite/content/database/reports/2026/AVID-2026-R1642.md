## Description

SSRF in gaizhenbiao/chuanhuchatgpt (CVE-2025-0188)

## Details

A Server-Side Request Forgery (SSRF) vulnerability was discovered in gaizhenbiao/chuanhuchatgpt version 20240914. The vulnerability allows an attacker to construct a response link by saving the response in a folder named after the SHA-1 hash of the target URL. This enables the attacker to access the response directly, potentially leading to unauthorized access to internal systems, data theft, service disruption, or further attacks such as port scanning and accessing metadata endpoints.

Reason for inclusion in AVID: The CVE describes an SSRF vulnerability in a software artifact named gaizhenbiao/chuanhuchatgpt, an AI chat-related application. This is a software component that could be used in building or deploying general-purpose AI systems and could be exploited to access internal resources, exfiltrate data, or disrupt services in AI pipelines. Therefore it affects the software supply chain of AI systems. The vulnerability is CVE-style with clear impact on security and safety. Sufficient evidence provided by CVE entry and CVSS data.
