## Description

The model rnj-1-instruct from Essential AI was evaluated by the Garak LLM Vulnerability scanner using the probe `encoding.InjectAscii85`.

## Details

The probe encoding.InjectAscii85 tests the injection of ASCII85 encoding into data. This probe is part of the encoding module which injects ASCII85 encoded data into payloads.

The LLM rnj-1-instruct was evaluated on this probe.
