## Description

OS Command Injection in mlflow/mlflow (CVE-2023-4033)

## Details

OS Command Injection in GitHub repository mlflow/mlflow prior to 2.6.0.

Reason for inclusion in AVID: CVE-2023-4033 describes an OS Command Injection in mlflow/mlflow prior to 2.6.0. MLflow is a core AI tooling/framework used to manage ML experiments, deployments, and pipelines, making this a software vulnerability affecting AI systems. The issue lies in a component (ML tooling) used to build/run AI workflows, fitting the general-purpose AI system supply chain. The CVE entry and references provide explicit vulnerability details and impact, fulfilling evidence requirements.
