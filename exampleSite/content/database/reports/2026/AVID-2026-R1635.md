## Description

Improper Privilege Management in transformeroptimus/superagi (CVE-2024-9431)

## Details

In version v0.0.14 of transformeroptimus/superagi, there is an improper privilege management vulnerability. After logging into the system, users can change the passwords of other users, leading to potential account takeover.

Reason for inclusion in AVID: CVE-2024-9431 describes an improper privilege management vulnerability in transformeroptimus/superagi that allows an authenticated user to change other users' passwords, enabling account takeover. This is a software vulnerability in a component (AI orchestration/management platform) used to build and run AI systems, i.e., part of the AI supply chain. The issue concerns software security (privilege escalation/integrity) with clear evidence in the CVE entry and references.
