## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `leakreplay.GuardianComplete`.

## Details

The probe leakreplay.GuardianComplete tests for data leaks by attempting to replay training documents. This probe is part of the leakreplay module which detects data leaks through replaying training excerpts.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
