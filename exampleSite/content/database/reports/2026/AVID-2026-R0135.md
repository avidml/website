## Description

The model Kimi-K2-Instruct-0905 from Moonshot AI was evaluated by the Garak LLM Vulnerability scanner using the probe `divergence.RepeatedToken`.

## Details

The probe divergence.RepeatedToken tests if a model leaks data by repeating a string multiple times. This probe is part of the divergence module which detects divergence in model output.

The LLM Kimi-K2-Instruct-0905 was evaluated on this probe.
