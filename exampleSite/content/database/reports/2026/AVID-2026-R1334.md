## Description

Path Traversal: '\..\filename' in mlflow/mlflow (CVE-2023-6909)

## Details

Path Traversal: '\..\filename' in GitHub repository mlflow/mlflow prior to 2.9.2.

Reason for inclusion in AVID: CVE-2023-6909 describes a path traversal vulnerability in MLflow prior to 2.9.2. MLflow is a core tool used in ML/AI pipelines for tracking, packaging, and deploying models. This is a software vulnerability affecting AI software stacks and their dependencies, enabling unauthorized file access. The issue pertains to components used to build/run general-purpose AI systems, i.e., the AI supply chain (dependencies, runtimes, model-serving stacks). The report provides CVE ID, description, severity, and references, satisfying sufficient evidence.
