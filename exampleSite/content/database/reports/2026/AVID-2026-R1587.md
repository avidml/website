## Description

Arbitrary File/Directory Deletion in aimhubio/aim (CVE-2024-6483)

## Details

A vulnerability in the `runs/delete-batch` endpoint of aimhubio/aim version 3.19.3 allows for arbitrary file or directory deletion through path traversal. The endpoint does not mitigate path traversal when handling user-specified run-names, which are used to specify log/metadata files for deletion. This can be exploited to delete arbitrary files or directories, potentially causing denial of service or data loss.

Reason for inclusion in AVID: CVE-2024-6483 describes a path traversal vulnerability in aimhubio/aim's runs/delete-batch endpoint, enabling arbitrary file/directory deletion. The affected component is used in AI experimentation/artifact management, i.e., part of AI pipelines. This aligns with AI-related software and the general-purpose AI software supply chain, and it is a CVE-style security vulnerability with clear exploit potential and evidence.
