## Description

IBM Watson CP4D Data Stores denial of service (CVE-2023-27540)

## Details

IBM Watson CP4D Data Stores 4.6.0 does not properly allocate resources without limits or throttling which could allow a remote attacker with information specific to the system to cause a denial of service.  IBM X-Force ID:  248924.

Reason for inclusion in AVID: CVE-2023-27540 targets IBM Watson CP4D Data Stores, a software component used within the CP4D AI platform. The vulnerability allows remote denial-of-service due to unbounded resource allocation, impacting the reliability of AI pipelines and deployments. This is a software vulnerability in a component used to build/deploy AI systems, relevant to the AI software supply chain.
