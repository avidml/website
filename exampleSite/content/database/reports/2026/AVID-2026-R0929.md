## Description

Vulnerability CVE-2021-45074

## Details

JFrog Artifactory before 7.29.3 and 6.23.38, is vulnerable to Broken Access Control, a low-privileged user is able to delete other known users OAuth token, which will force a reauthentication on an active session or in the next UI session.

Reason for inclusion in AVID: CVE-2021-45074 describes a broken access control vulnerability in JFrog Artifactory affecting OAuth tokens; Artifactory is a widely-used artifact/repository tool in software supply chains and ML pipelines, so this is a software supply chain vulnerability relevant to general-purpose AI systems; it is a security vulnerability with clear evidence (CVE entry, CVSS details, references).
