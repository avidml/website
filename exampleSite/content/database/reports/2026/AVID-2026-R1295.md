## Description

Authentication Bypass by Primary Weakness in mintplex-labs/anything-llm (CVE-2023-4898)

## Details

Authentication Bypass by Primary Weakness in GitHub repository mintplex-labs/anything-llm prior to 0.0.1.

Reason for inclusion in AVID: CVE-2023-4898 describes a remote authentication bypass in mintplex-labs/anything-llm (prior to 0.0.1). The affected project is an AI-focused library used in AI model tooling/workflows, i.e., a software component commonly involved in building/deploying GP AI systems. This constitutes a software supply-chain vulnerability affecting dependencies/runtimes in AI stacks. The CVE is security-related with public references (NVD entry, commit), providing sufficient signal.
