## Description

Vulnerability CVE-2024-21949

## Details

Improper validation of user input in the NPU driver could allow an attacker to provide a buffer with unexpected size, potentially leading to system crash.

Reason for inclusion in AVID: CVE-2024-21949 describes improper input validation in the AMD NPU driver (part of AMD Ryzen AI Software). This is a software component used to accelerate AI workloads and run AI systems, i.e., a component in the AI software stack. The issue is a vulnerability (potential crash) in a driver, affecting availability, and relates to software supply chain components (drivers, AI acceleration stack). The report provides CVSS details and references, giving sufficient signal for classification.
