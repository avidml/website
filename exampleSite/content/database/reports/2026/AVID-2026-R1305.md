## Description

Segfault in paddle.nextafter (CVE-2023-52302)

## Details

Nullptr in paddle.nextafter in PaddlePaddle before 2.6.0. This flaw can cause a runtime crash and a denial of service.

Reason for inclusion in AVID: CVE-2023-52302 describes a software vulnerability in PaddlePaddle (a deep learning framework) that can cause a runtime crash and denial of service. This affects AI models/pipelines that rely on PaddlePaddle, i.e., AI systems software stacks. It involves a dependency/framework component (not hardware), fitting the general-purpose AI software supply chain scope. The CVE details provide explicit impact and evidence (null pointer dereference, segfault, DoS), satisfying the security/safety criteria.
