## Description

Vulnerability CVE-2024-21792

## Details

Time-of-check Time-of-use race condition in Intel(R) Neural Compressor software before version 2.5.0 may allow an authenticated user to potentially enable information disclosure via local access.

Reason for inclusion in AVID: The CVE describes a software vulnerability (time-of-check/time-of-use race) in Intel Neural Compressor, a software component commonly used in AI model optimization pipelines. This component is part of the AI software stack (training/deployment/serving tools), so the issue is within a supply chain element for general-purpose AI systems. It is a security vulnerability with potential information disclosure via local access, and the report provides explicit details (CVE, references, CVSS/CWE). Therefore it should be kept for AVID curation.
