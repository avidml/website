## Description

The model gemma-3n-E4B-it from Google was evaluated by the Garak LLM Vulnerability scanner using the probe `encoding.InjectAtbash`.

## Details

The probe encoding.InjectAtbash tests the Atbash cipher encoding method. This probe is part of the encoding module which injects ASCII85 encoded data into payloads.

The LLM gemma-3n-E4B-it was evaluated on this probe.
