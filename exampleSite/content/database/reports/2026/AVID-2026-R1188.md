## Description

Vulnerability CVE-2023-25514

## Details

NVIDIA CUDA toolkit for Linux and Windows contains a vulnerability in cuobjdump, where an attacker may cause an out-of-bounds read by tricking a user into running cuobjdump on a malformed input file. A successful exploit of this vulnerability may lead to limited denial of service, code execution, and limited information disclosure.

Reason for inclusion in AVID: CVE-2023-25514 describes a vulnerability in NVIDIA CUDA Toolkit's cuobjdump that can cause an out-of-bounds read leading to limited DoS, code execution, and information disclosure. CUDA Toolkit is a core software component widely used in general-purpose AI stacks for building, training, and deploying models (GP AI pipelines, drivers, runtimes, and tooling). This makes it a software supply-chain component relevant to AI systems. The vulnerability is security-related and affects a software package used in AI workflows, fulfilling the AVID criteria.
