## Description

The model Mistral-Small-24B-Instruct-2501 from Mistral was evaluated by the Garak LLM Vulnerability scanner using the probe `encoding.InjectBase32`.

## Details

The probe encoding.InjectBase32 tests the injection of Base32 encoded data. This probe is part of the encoding module which injects ASCII85 encoded data into payloads.

The LLM Mistral-Small-24B-Instruct-2501 was evaluated on this probe.
