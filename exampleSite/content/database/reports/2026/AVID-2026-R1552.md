## Description

TorchGeo Remote Code Execution Vulnerability (CVE-2024-49048)

## Details

TorchGeo Remote Code Execution Vulnerability

Reason for inclusion in AVID: CVE-2024-49048 describes a remote code execution vulnerability in TorchGeo, a Python library used in ML/AI workflows (e.g., geospatial data processing with PyTorch). This directly concerns software used to build/train/deploy AI systems, representing a supply chain vulnerability in AI software stacks (dependencies/frameworks). It is a security/safety vulnerability (RCE) with available CVE reference and sources. The evidence in the report (CVE ID, description, and references) supports classification as a software supply chain vulnerability affecting AI pipelines.
