## Description

External Control of File Name or Path in h2oai/h2o-3 (CVE-2023-6569)

## Details

External Control of File Name or Path in h2oai/h2o-3

Reason for inclusion in AVID: CVE-2023-6569 describes external control of a file name or path in h2oai/h2o-3, a machine learning framework. This is a software vulnerability within a component used to build/train/deploy AI systems, with remote exploitation potential (network-based) and impact on availability and integrity. It directly concerns AI software stacks and their pipelines, not hardware/firmware. The AVID candidate includes official CVE context and references, providing sufficient evidence for inclusion.
