## Description

TensorFlow has Floating Point Exception in AudioSpectrogram (CVE-2023-25666)

## Details

TensorFlow is an open source platform for machine learning. Prior to versions 2.12.0 and 2.11.1, there is a floating point exception in AudioSpectrogram. A fix is included in TensorFlow version 2.12.0 and version 2.11.1.

Reason for inclusion in AVID: CVE-2023-25666 describes a vulnerability in TensorFlow (AudioSpectrogram) that can cause a crash/denial of service. TensorFlow is a core AI framework used in ML pipelines; this constitutes a software supply-chain vulnerability affecting AI systems. The issue affects a widely-used component and has a CVE with a fix in a newer version; evidence provided by NVD and GitHub advisories supports classification.
