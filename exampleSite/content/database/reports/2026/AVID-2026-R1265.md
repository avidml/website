## Description

FPE in paddle.linalg.eig (CVE-2023-38677)

## Details

FPE in paddle.linalg.eig in PaddlePaddle before 2.6.0. This flaw can cause a runtime crash and a denial of service.

Reason for inclusion in AVID: CVE-2023-38677 describes a flaw in PaddlePaddle (paddle.linalg.eig) that can cause a runtime crash and DoS. PaddlePaddle is a general-purpose AI framework, so the issue affects software components used to build, train, and deploy AI systems. This is a vulnerability in a software supply chain component (AI framework) with explicit vulnerability behavior (crash/DoS). The report provides the CVE, description, impact, and references, offering sufficient signals for classification.
